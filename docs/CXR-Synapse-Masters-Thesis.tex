\chapter*{ანოტაცია}
\addcontentsline{toc}{chapter}{ანოტაცია}
გულ-მკერდის რენტგენოგრამების (CXR) მრავალნიშნიანი კლასიფიკაცია თანამედროვე სამედიცინო ხელოვნური ინტელექტის ერთ-ერთ უმნიშვნელოვანეს გამოწვევას წარმოადგენს, რაც გამოწვეულია კლასთა უკიდურესი დისბალანსითა და მოდელების ზე-თავდაჯერებულობით. წინამდებარე ნაშრომში წარმოდგენილია კლინიკურად ორიენტირებული არქიტექტურა, რომელიც პირველად აერთიანებს LISA (Language-Invariant Semantic Anchoring) რეგულარიზაციას BioViL-T ენობრივი პრიორებით, PaQ (Pathology-as-Query) სივრცითი ჯვარედინი ყურადღების მექანიზმს ორგანზომილებიანი როტაციული პოზიციური ემბედინგებით (2D RoPE) და ბინომიალურ სუბიექტურ ლოგიკას ეპისტემური გაურკვევლობის ზუსტი რაოდენობრივი დეკუპლირებისთვის. ონტოლოგიური ცოდნის ასეთი სინთეზი ხელს უწყობს მოდელის მიერ ანატომიური ნიშნების გააზრებულ ამოცნობას და გამორიცხავს ზედაპირულ ასოციაციებზე სწავლებას (Shortcut Learning).
ჩატარებულმა ექსპერიმენტებმა ChestMNIST მონაცემთა ბაზაზე აჩვენა, რომ შემოთავაზებული მოდელი წარმატებით არღვევს ტრადიციულ რეპრეზენტაციულ ბარიერებს და აღწევს მაკრო-საშუალო AUC 0.7889-ს. პოსტ-ჰოკ კლასობრივი ასიმეტრიული იზოტონური კალიბრაციის (AIR) გამოყენებით, კალიბრაციის საშუალო შეცდომა (Mean ECE) შემცირდა 0.0117-მდე (1.17\%). ნაშრომში ინტეგრირებულია გაურკვევლობით მართული კონფორმალური რისკის კონტროლის სისტემა (CRC), რომლითაც მიღწეულია კლასთა საშუალო კონფორმალური დაფარვა 52.0\% და პაციენტის დონის დაფარვა 61.7\%, ხოლო პროგნოზების საშუალო სიმრავლის ზომა შეადგენს 2.95-ს. შედეგების ვალიდურობა დადასტურებულია 2000-ჯერადი Bootstrap რესემპლინგითა და Blake2b ფორენზიკული დედუპლიკაციით. აღნიშნული არქიტექტურა ქმნის საიმედო საფუძველს სანდო ხელოვნური ინტელექტის (Trustworthy AI) სისტემების მაღალი რისკის მქონე კლინიკურ პრაქტიკაში უსაფრთხო დასანერგად. ნაშრომში დეტალურად გაწერილი ოპერაციული პროტოკოლები და ეთიკური ჩარჩო განსაზღვრავს სისტემის შემდგომი მულტიცენტრული გარე ვალიდაციისა და რეალურ სამედიცინო გარემოში უსაფრთხო ინტეგრაციის გზებს.

\begin{english}
\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract (in English)}

\noindent\textbf{Title:} Multi-Label Classification of Chest X-Rays Using LISA and PaQ Algorithms with Conformal Safety Guarantees \\
\noindent\textbf{Author:} Nika Natsvlishvili \\
\noindent\textbf{Supervisor:} Davit Datuashvili, PhD in Computer Science \\

\vspace{1.5em}
Multi-label classification of Chest X-rays (CXR) represents a critical challenge in modern medical AI due to extreme class imbalance and model overconfidence. This thesis presents a clinically oriented architecture that, for the first time, integrates LISA (Language-Invariant Semantic Anchoring) regularization with BioViL-T linguistic priors, a PaQ (Pathology-as-Query) spatial cross-attention mechanism on 2D rotary positional embedding grids (2D RoPE), and Binomial Subjective Logic to quantitatively decouple epistemic uncertainty. This ontological synthesis encourages the model to learn meaningful anatomical features, mitigating shortcut learning from visual artifacts.

Experiments on ChestMNIST demonstrate that the proposed model successfully breaks traditional representational barriers, achieving a macro-averaged multi-label AUC of 0.7889. Post-hoc class-wise asymmetric isotonic regression (AIR) reduced the Mean Expected Calibration Error (ECE) to 0.0117 (1.17\%). Incorporating an Uncertainty-Gated Conformal Risk Control (CRC) system yields a class-average conformal coverage of 52.0\%, patient-level coverage of 61.7\%, and a mean prediction set size of 2.95. Outcomes are validated using 2000-fold bootstrap resampling and Blake2b forensic deduplication. This architecture provides a reliable foundation for safely deploying Trustworthy AI systems in high-risk clinical practice. Finally, the detailed operational protocols and ethical frameworks outline pathways for subsequent multi-centric external validation and safe real-world integration.

\vspace{1.5em}

\noindent\textbf{Keywords:} Chest X-ray, multi-label classification, Pathology-as-Query, LISA, calibration, Evidential Deep Learning, Conformal Prediction, 2D RoPE, Subjective Logic.
\end{english}

\section{პრობლემის აქტუალობა}
გულ-მკერდის რენტგენოგრაფია (CXR) კლინიკურ პრაქტიკაში ყველაზე მასობრივი დიაგნოსტიკური მეთოდია, რაც მის ავტომატიზებულ ანალიზს განსაკუთრებულ მნიშვნელობას ანიჭებს კლინიკური ტრაიაჟისა და დიაგნოსტიკური ხარისხის გაუმჯობესებისთვის. მიუხედავად ღრმა სწავლების (Deep Learning) მოდელების პროგრესისა, მრავალნიშნიანი კლასიფიკაცია კვლავ კრიტიკულ გამოწვევად რჩება სამი ძირითადი ფაქტორის გამო: 
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{კლასთა უკიდურესი დისბალანსი}, სადაც იშვიათი პათოლოგიები (მაგ. თიაქარი — Hernia) 100-ჯერ უფრო იშვიათია გავრცელებულ კლინიკურ შემთხვევებზე (მაგ. ინფილტრაცია — Infiltration);
    \item \textbf{წარმოდგენების დრიფტი (Representational Drift)}, როდესაც გაყინული (frozen) ვიზუალური ენკოდერები კარგავენ სპეციფიკური პათოლოგიების დისკრიმინაციული ამოცნობის უნარს ლოკალური მახასიათებლების გადაფარვის გამო;
    \item \textbf{მოდელის ზე-თავდაჯერებულობა (Overconfidence)}, რაც იწვევს არასწორ, მაგრამ ალბათურად მაღალი ნდობით გაცემულ მცდარ დიაგნოზებს.
\end{enumerate}

ტრადიციული კონვოლუციური ქსელებისგან (CNN) განსხვავებით, რომლებიც ეყრდნობიან ლოკალურ რეცეპტულ ველებს, შემოთავაზებული მოდელი იყენებს Vision Transformer (ViT) არქიტექტურის პრინციპებს. როგორც დოსოვიცკი და სხვები (2021) აღნიშნავენ, ViT-ს გააჩნია გაცილებით ნაკლები გამოსახულებაზე სპეციფიკური ინდუქციური ბიასი, რაც საშუალებას აძლევს სისტემას, დიდი მოცულობის მონაცემებზე (Large-scale pre-training) წვრთნისას ისწავლოს უფრო ზოგადი, ძლიერი და მდგრადი რეპრეზენტაციები.\footnote{დოსოვიცკი, ა., ბეიერი, ლ., კოლესნიკოვი, ა. და სხვა, „An Image is Worth 16x16 Words: Transformers for Image Recognition at Scale“, International Conference on Learning Representations (ICLR), 2021.}

სამედიცინო დიაგნოსტიკაში მოდელის მხოლოდ მაღალი სიზუსტე (Accuracy) კლინიკურად არასაკმარისია, რადგან, როგორც გუომ და სხვებმა (2017) დაამტკიცეს, თანამედროვე ღრმა ნეირონული ქსელები ხშირად ცუდად კალიბრირებულნი არიან (Poorly Calibrated). ეს ნიშნავს, რომ მოდელის მიერ გაცემული ალბათობა არ ასახავს რეალურ ემპირიულ დიაგნოსტიკურ ჭეშმარიტებას, რაც კლინიკური გადაწყვეტილების მიღებისას კრიტიკული რისკის წყაროა.\footnote{გუო, ჩ., პლეისი, ჯ., სანი, ი. და სხვა, „On Calibration of Modern Neural Networks“, Proceedings of the 34th International Conference on Machine Learning (ICML), 2017.}

როგორც ვანგი და სხვები (2026) აღნიშნავენ თავიანთ უახლეს კვლევაში, ტრადიციული Vision-Language მოდელები განიცდიან სემანტიკურ კოლაფსს, რადგან სამედიცინო ატრიბუტები აღქმულია როგორც უბრალო ტექსტური მიმდევრობა და არა როგორც დისკრიმინაციული კლინიკური ერთეულები. შემოთავაზებული მოდელი ამ პრობლემას პასუხობს ონტოლოგიური LISA (Language-Invariant Semantic Anchoring) და PaQ (Pathology-as-Query) მექანიზმებით.\footnote{ვანგი, ჰ., ქსუ, ქს., ვანგი, ლ. და სხვა, „AttTok: Marrying Attribute Tokens with Generative Pre-trained Vision-Language Models towards Medical Image Understanding“, International Conference on Learning Representations (ICLR), 2026.}

სამედიცინო ხელოვნური ინტელექტისთვის კრიტიკულია არა მხოლოდ დიაგნოზის დასმა, არამედ იმის აღიარება, როდესაც მას არ გააჩნია საკმარისი მტკიცებულება პროგნოზისთვის. როგორც სენსოი და სხვები (2018) აღნიშნავენ, ტრადიციული მოდელები მოკლებულნი არიან „I do not know“ (მე არ ვიცი) პასუხის გაცემის შესაძლებლობას. შემოთავაზებული მოდელი ამ პრობლემას წყვეტს ევიდენციური პრინციპების ინტეგრირებით, რაც საშუალებას იძლევა მოხდეს გაურკვევლობის მათემატიკური დეკუპლირება ბინომიალური საკლასო სუბიექტური ლოგიკის (Subjective Logic) თეორიის საფუძველზე.\footnote{სენსოი, მ., კაპლანი, ლ. მ., კანდემირი, მ., „Evidential Deep Learning to Quantify Classification Uncertainty“, Advances in Neural Information Processing Systems (NeurIPS), 2018.} შესაბამისად, მხოლოდ მაღალი AUC მაჩვენებელი საკმარისი არ არის; აუცილებელია მოდელის კალიბრაციისა და პროგნოზის გაურკვევლობის (Uncertainty) სტატისტიკურად დასაბუთებული შეფასება. როგორც ბალტრუშატი და სხვები (2019) აღნიშნავენ, დიდ ბრიტანეთში ჩატარებულმა კლინიკურმა აუდიტმა აჩვენა, რომ 23,000-ზე მეტი რენტგენოგრამა დარჩა ფორმალური შეგასების გარეშე, რამაც ფატალური შედეგები გამოიწვია.\footnote{ბალტრუშატი, ი. მ., ნიკიში, ჰ., გრასი, მ. და სხვა, „Comparison of Deep Learning Approaches for Multi-Label Chest X-Ray Classification“, Scientific Reports, ტომი 9, ნომერი 6381, 2019.} ეს ხაზს უსვამს ავტომატიზებული ტრაიაჟის (Triage) სისტემების კრიტიკულ საჭიროებას მაღალი სანდოობის გარანტიებით.

სამედიცინო დიაგნოსტიკაში, სადაც ალგორითმები ხშირად „შავი ყუთის“ პრინციპით მუშაობენ, უდიდეს საფრთხეს წარმოადგენს ე.წ. „ჩუმი ჩავარდნები“ (Silent Failures). როგორც ბეიტსი და სხვები (2021) აღნიშნავენ, დიაგნოსტიკური სიზუსტე მარტო არ არის საკმარისი სანდო გადაწყვეტილების მისაღებად.\footnote{ბეიტსი, ს., ანგელოპულოსი, ა. ნ., ლეი, ლ. და სხვა, „Distribution-Free, Risk-Controlling Prediction Sets“, Journal of the ACM (JACM), ტომი 68, ნომერი 6, 2021.} შემოთავაზებული მოდელი ამ პრობლემას პასუხობს გაურკვევლობით მართული კონფორმალური პროგნოზირების ჩარჩოს დანერგვით, რაც უზრუნველყოფს, რომ მოდელის შეცდომის ალბათობა იყოს მკაცრად კონტროლირებული მომხმარებლის მიერ დაწესებულ ზღვარში. როგორც დაგნავი და სხვები (2026) აღნიშნავენ თავიანთ ნაშრომში DCG-Net, ტრადიციული სალიენსური რუკები (Saliency-based explanations) ხშირად არასაკმარისია მაღალი რისკის კლინიკური გადაწყვეტილებებისთვის, რადგან ისინი მხოლოდ გამოსახულების რეგიონებს გამოყოფენ, მაგრამ ვერ ხსნიან მათ კლინიკურ ლოგიკას. შემოთავაზებული მოდელი ამ პრობლემას წყვეტს PaQ მექანიზმის საშუალებით, რაც პირდაპირ ეხმიანება DCG-Net-ის მიერ შემოთავაზებულ სტრუქტურირებულ მსჯელობას.

კვლევის ობიექტად შერჩეულია ChestMNIST მონაცემთა ბაზა, რომელიც წარმოადგენს MedMNIST v2-ის სტანდარტიზებული ბიოსამედიცინო ბენჩმარკის ნაწილს.\footnote{იანგი, ჯ., ში, რ., ვეი, დ. და სხვა, „MedMNIST v2: A Large-Scale Lightweight Benchmark for 2D and 3D Biomedical Image Classification“, Scientific Data, ტომი 10, ნომერი 1, 2023.} როგორც იანგი და სხვები აღნიშნავენ, MedMNIST-ის მიზანია „MNIST-ის მსგავსი“ გამარტივებული, მაგრამ კლინიკურად რეალური მონაცემების მიწოდება, რაც საშუალებას იძლევა AI მოდელების სწრაფი პროტოტიპირებისა და ობიექტური შედარებისთვის.

\section{კვლევის მიზანი}
კვლევის მიზანია შეიქმნას და შეფასდეს \textbf{შემოთავაზებული მოდელის} ერთიანი მეთოდოლოგიური ჩარჩო, რომელიც ELIXR-C/CXR-Foundation ვიზუალურ რეპრეზენტაციებს აერთიანებს RadLex-ით განპირობებულ ლეიბლ-გრაფულ მიზეზობრიობასთან (LISA და PaQ), კლასობრივად ბალანსირებულ ასიმეტრიულ ოპტიმიზაციასთან და გაურკვევლობით მართულ კონფორმალურ გარანტიებთან. 

აღნიშნული მიზანი ფორმულირებულია ალიფერისა და სიმონის მიერ შემოთავაზებული \textit{Biomedical AI/ML Trust and Acceptance} ჩარჩოს მიხედვით.\footnote{სიმონი, ჯ. ჯ., ალიფერისი, ქ., რედ., „Artificial Intelligence and Machine Learning in Health Care and Medical Sciences: Best Practices and Pitfalls“, Springer, Cham, 2024. გვ-გვ. 15–16.} კვლევა ფოკუსირებულია ნდობის ორ უმთავრეს განზომილებაზე:
\begin{enumerate}
    \item \textbf{Scientific and Technical Trust}: ალბათური სიზუსტე და კალიბრაცია, რაც მოდელში რეალიზებულია კლასობრივად ასიმეტრიული იზოტონური კალიბრაციით (AIR) და მოსალოდნელი კალიბრაციის შეცდომის (ECE) მინიმიზაციით;
    \item \textbf{Health System Trust}: კლინიკური უსაფრთხოება, რაც უზრუნველყოფილია მათემატიკურად დაზღვეული კონფორმალური გარანტიებით (CRC) და ეპისტემური გაურკვევლობის ფილტრაციით (Abstention Gating) კლინიკურად რელევანტურ პირობებში.
\end{enumerate}

\section{კვლევის ამოცანები}
დასმული მიზნის მისაღწევად განისაზღვრა შემდეგი ამოცანები:
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{Pathology-as-Query (PaQ)} სივრცითი ყურადღების მექანიზმის იმპლემენტაცია **ორგანზომილებიანი როტაციული პოზიციური ემბედინგების (2D RoPE)** გამოყენებით, რაც უზრუნველყოფს სივრცითი კოორდინატების უზადო პროექციას;
    \item \textbf{Language-Invariant Semantic Anchoring (LISA)} რეგულარიზაციის დანერგვა RadLex ონტოლოგიაზე დაფუძნებული სემანტიკური ტოპოლოგიის შესანარჩუნებლად და მულტიმოდალური ტრანსფორმაციისთვის;
    \item კლასობრივად ბალანსირებული ასიმეტრიული დანაკარგისა (**Class-Balanced Asymmetric Loss - CB-ASL**) და **LISA Semantic Anchor Loss**-ის ჰიბრიდული ოპტიმიზაციის ჩარჩოს შემუშავება კალიბრაციის შეცდომის (ECE) მინიმიზაციისა და ეპისტემური გაურკვევლობის რაოდენობრივი შეფასებისთვის;
    \item გაურკვევლობის შეფასების მექანიზმის აგება **Deep Ensemble** და **Stochastic Weight Averaging (SWA)** მიდგომების სინერგიით, ლოგიტების სივრცეში ალბათური ჰარმონიზაციითა და ეპისტემური ენტროპიის დეკუპლირებით;
    \item გაურკვევლობით მართული კონფორმალური რისკის კონტროლის ჩარჩოს (**Uncertainty-Gated Adaptive Conformal Predictor - CRC**) დანერგვა, რაც უზრუნველყოფს პათოლოგიების გარანტირებულ დაფარვას ეპისტემური ფილტრაციის ზღვარის პირობებში;
    \item შედეგების სტატისტიკური ვალიდაცია 2000-ჯერადი Bootstrap ნდობითი ინტერვალებითა და დაწყვილებული არაპარამეტრული მნიშვნელოვნების ტესტებით.
\end{enumerate}

\section{კვლევის მეთოდოლოგია}
კვლევის მეთოდოლოგია უზრუნველყოფს მიღებული დასკვნების მათემატიკურ სანდოობასა და კლინიკურ გამჭვირვალობას:
\begin{itemize}
    \item \textbf{შედარებითი სამეცნიერო ანალიზი:} სისტემური მიმოხილვა გამოიყენება პრობლემის ფორმალიზაციისა და SOTA (2024-2026) მეთოდების იდენტიფიცირებისთვის, რაც საფუძველს უქმნის ახალი მოდელის პოზიციონირებას.
    \item \textbf{Representation Learning კონტროლირებადი აბლაციებით:} მოდელის კომპონენტების (LISA, PaQ, CB-ASL Loss) წვლილი იზომება იზოლირებული ექსპერიმენტებით. ეს მეთოდი რელევანტურია, რადგან აჩვენებს, რომელი მოდული ქმნის რეალურ გაუმჯობესებას კალიბრაციასა და დისკრიმინაციულ სიზუსტეში.
    \item \textbf{სტატისტიკური დასკვნითი ანალიზი:} Bootstrap ნდობითი ინტერვალები და დაწყვილებული ბუთსტრეპ მნიშვნელოვნების ტესტები (Paired Bootstrap Test) გამოიყენება შედეგების შემთხვევითი ვარიაციისგან გასამიჯნად. მეთოდის სანდოობა ეფუძნება 2000-ჯერ განმეორებად ნიმუშებზე შეფასებას.
    \item \textbf{კალიბრაციისა და Conformal რისკის შეფასება:} ECE (Expected Calibration Error) და Conformal Coverage მეტრიკები გამოიყენება პროგნოზის სანდოობის შესაფასებლად. ეს პირდაპირ პასუხობს „Trustworthy AI“-ს მიზანს: სისტემამ უნდა წარმოადგინოს სანდოობის ინტერპრეტირებადი ხარისხი.
\end{itemize}

\section{ლიტერატურის მიმოხილვა}
სამედიცინო გამოსახულებების ანალიზში მულტიმოდალური სისტემების გამოყენება (მაგალითად Google-ის ELIXR-C) თანამედროვე კვლევების წამყვანი მიმართულებაა. თუმცა, არსებული ლიტერატურის ანალიზი აჩვენებს, რომ კლინიკური კონცეფციების კორელაციები ხშირად იგნორირებულია, რაც იწვევს სემანტიკურ კოლაფსს. უახლესი მოდელები, როგორიცაა DCG-Net, პრობლემას გრაფული მიდგომებით პასუხობენ. ევიდენციური ღრმა სწავლების (EDL) პრინციპები, რომლებიც სენსოის მიერ იქნა შემოტანილი, შესაძლებლობას იძლევა მოხდეს გაურკვევლობის რაოდენობრივი შეფასება, თუმცა მათი ინტეგრაცია კონფორმალურ უსაფრთხოებასთან მრავალლეიბლიან CXR ამოცანებში დღემდე ნაკლებად არის შესწავლილი, რაც ქმნის წინამდებარე კვლევის სამეცნიერო ნიშას.

\section{ნაშრომის აგებულების მოკლე აღწერა}
სამაგისტრო ნაშრომი შედგება შესავლის, შვიდი თავისა და ათი დანართისგან. პირველი თავი ეთმობა პრობლემის აქტუალობასა და მეთოდოლოგიას. მეორე თავში წარმოდგენილია ლიტერატურის სისტემური მიმოხილვა. მესამე თავი აღწერს შემოთავაზებულ LISA და PaQ არქიტექტურას. მეოთხე თავი ასახავს ექსპერიმენტულ პროტოკოლს, ხოლო მეხუთე თავი ეთმობა მიღებული რაოდენობრივი და ხარისხობრივი შედეგების ანალიზს. მეექვსე და მეშვიდე თავებში წარმოდგენილია მიღებული შედეგების კლინიკური ინტერპრეტაცია, დისკუსია და საბოლოო დასკვნები.

\section{სამედიცინო გამოსახულების ანალიზის ტრადიციული მეთოდები}
სამედიცინო ვიზუალიზაციაში ტრადიციული ანალიზი დიდწილად ეყრდნობოდა წესებზე დაფუძნებულ და ხელით მორგებულ (Hand-crafted) მიდგომებს. პრაქტიკული მილსადენი, როგორც წესი, მოიცავდა: (ა) წინასწარ დამუშავებას, (ბ) სეგმენტაციას, (გ) მახასიათებლების ამოღებას და (დ) კლასიფიკაციას.

\textbf{წინასწარი დამუშავება.}
ყველაზე გავრცელებული ოპერაციებია ხმაურის შემცირება (Gaussian/Median ფილტრები), კონტრასტის ლოკალური გაძლიერება (CLAHE) და ინტენსივობის ნორმალიზაცია. მიუხედავად იმისა, რომ ეს ნაბიჯები აუმჯობესებს ვიზუალურ ხარისხს, ისინი ვერ უზრუნველყოფენ მოდელის მდგრადობას დომენურ ცვლადობაზე (Domain Shift), რაც თანამედროვე ფუნდამენტური მოდელების (Foundation Models) მთავარი უპირატესობაა.

\textbf{სეგმენტაცია.}
კლასიკური მეთოდები (ზღურბლოვანი ალგორითმები, კიდეების დეტექცია, Region Growing, Watershed) ეფექტურია მაღალკონტრასტულ სცენარებში, თუმცა მათი სიზუსტე მკვეთრად მცირდება რენტგენოგრაფიული გამოსახულებების ჰეტეროგენულობის, ანატომიური სტრუქტურების გადაფარვისა და ვიზუალური არტეფაქტების პირობებში. სამედიცინო გამოსახულებების ბუნდოვანი საზღვრები ტრადიციული სეგმენტაციისთვის გადაულახავ ბარიერს წარმოადგენს.

\textbf{ხელით ინჟინერირებული მახასიათებლები.}
ტრადიციულად, ტექსტურული (GLCM), ფორმითი (Shape descriptors) და ინტენსივობითი დესკრიპტორები გამოიყენებოდა SVM, Random Forest ან ლოგისტიკური რეგრესიის კლასიფიკატორებთან ერთად. ამ მიდგომების მთავარი შეზღუდვა არის მახასიათებლების ხელით შერჩევაზე (Feature Engineering) მაღალი დამოკიდებულება, რაც ვერ უზრუნველყოფს მოდელის მასშტაბირებადობას და ვერ აღქვამს რთულ პათოლოგიურ პატერნებს.

\textbf{შეჯამება.}
ტრადიციულმა მეთოდებმა შექმნა ისტორიული საფუძველი კომპიუტერული დიაგნოსტიკის (CAD) სისტემებისთვის, თუმცა მრავალლეიბლიან კლინიკურ ამოცანებში მათი ეფექტუყოფა შეზღუდულია. შესაბამისად, თანამედროვე კვლევა სრულად გადავიდა მონაცემებზე დაგებულ ღრმა სწავლის მოდელებზე.

\section{ვიზუალურ-ტექსტური (VLM) ფუნდამენტური მოდელები}
2024-2026 წლების კვლევებმა აჩვენა, რომ სამედიცინო გამოსახულებების ანალიზი გადავიდა იზოლირებული ენკოდერებიდან მულტიმოდალურ სისტემებზე. მოდელები, როგორიცაა Google-ის ELIXR-C, იყენებენ მილიონობით წყვილ რენტგენოგრამასა და რადიოლოგიურ რეპორტს, რათა ისწავლონ მაღალი დონის სემანტიკური რეპრეზენტაციები.\footnote{ქსუ, ს., იანგი, ლ., კელი, ს. და სხვა, „ELIXR: Towards a General Purpose X-ray Artificial Intelligence System Through Alignment of Large Language Models and Radiology Vision Encoders“, arXiv preprint arXiv:2308.01317, 2024.} თუმცა, ამ მოდელების გაყინული გამოყენება (Frozen Embeddings) იწვევს კლინიკურ ბუნდოვანებას და სემანტიკურ დრიფტს, რის გადასაჭრელადაც შემოთავაზებულ მოდელში დაინერგა LISA (Language-Invariant Semantic Anchoring) რეგულარიზაციის მექანიზმი.

ImageNet-ზე გაწვრთნილი მოდელებისგან განსხვავებით, წინამდებარე კვლევაში გამოყენებულია Google-ის მიერ შემოთავაზებული ELIXR-C (CXR-Foundation v2). როგორც Xu და სხვები (2024) აღნიშნავენ, ELIXR-C არის ენობრივ-ვიზუალური კონტრასტული სწავლებით (CLIP-style) მიღებული ფუნდამენტური მოდელი, რომელიც სპეციალურად რენტგენოლოგიური რეპორტების სემანტიკაზეა ოპტიმიზებული.\footnote{ქსუ, ს., იანგი, ლ., კელი, ს. და სხვა, „ELIXR: Towards a General Purpose X-ray Artificial Intelligence System Through Alignment of Large Language Models and Radiology Vision Encoders“, arXiv preprint arXiv:2308.01317, 2024.} ეს უზრუნველყოფს, რომ მოდელის მიერ გენერირებული 1376-განზომილებიანი ემბედინგები უკვე ფლობენ ანატომიურ და პათოლოგიურ პრიორებს.

ბოლო დროის უახლესი კვლევები, როგორიცაა DCG-Net (2026), აკრიტიკებენ კლასიკურ Concept Bottleneck მოდელებს (CBM) იმის გამო, რომ ისინი უგულებელყოფენ კლინიკურ კონცეფციებს შორის არსებულ კორელაციებს. დაგნავის მიერ შემოთავაზებული გრაფული მიდგომა ადასტურებს, რომ დიაგნოსტიკური სიზუსტე იზრდება, როდესაც მოდელი ითვალისწინებს პათოლოგიათა თანაარსებობის (Co-occurrence) სტრუქტურას. შემოთავაზებულ არქიტექტურაში ეს სინერგია მიღწეულია კოსინუსური სემანტიკური მსგავსებისა და ემპირიული კლინიკური თანაავადობის შერწყმით, რაც მოდელის გრაფულ მატრიცას (\texttt{adjacency\_mask}) დაედო საფუძვლად.

\section{მანქანური სწავლება სამედიცინო გამოსახულებისთვის}
მანქანურმა სწავლებამ (ML) სამედიცინო ანალიზი გადაიყვანა ფიქსირებული წესებიდან სტატისტიკურ სწავლაზე. ძირითადი პარადიგმებია:
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{ზედამხედველობითი სწავლება (Supervised Learning)}: ანოტირებულ მონაცემებზე დაყრდნობით პათოლოგიების კლასიფიკაცია.
    \item \textbf{ზედამხედველობის გარეშე სწავლება (Unsupervised Learning)}: ანომალიების აღმოჩენა და კლასტერიზაცია წინასწარი ნიშნულების გარეშე.
    \item \textbf{ნახევრად ზედამხედველობითი სწავლება (Semi-supervised Learning)}: მცირე ანოტირებული ნაკრებისა და დიდი მოცულობის არალეიბლიანი მონაცემების კომბინაცია.
\end{enumerate}

კლასიკურ ML მილსადენში წარმატება კრიტიკულად იყო დამოკიდებული მახასიათებლების საინჟინრო დამუშავებაზე (Feature Engineering). მიუხედავად იმისა, რომ ეს მეთოდები კვლავ გამოიყენება საბაზისო შედარებებისთვის, რთული ვიზუალური პატერნების აღქმის უნარით ისინი მნიშვნელოვნად ჩამორჩებიან ღრმა ნეირონულ ქსელებს.\footnote{ჰასტი, თ., ტიბშირანი, რ., ფრიდმანი, ჯ. ჰ., „The Elements of Statistical Learning: Data Mining, Inference, and Prediction“, Springer, 2009.}

\section{ღრმა სწავლება სამედიცინო გამოსახულებისთვის}
ღრმა სწავლების (Deep Learning) მთავარი უპირატესობაა End-to-End სწავლება, სადაც მოდელი პირდაპირ პიქსელური დონიდან სწავლობს იერარქიულ და აბსტრაქტულ რეპრეზენტაციებს. სამედიცინო სეგმენტაციის არქიტექტურებში სემანტიკური ნაპრალის პრობლემა ენკოდერსა და დეკოდერს შორის ერთ-ერთ მთავარ გამოწვევად ითვლება. Zhou et al. (2018) მიერ შემოთავაზებული UNet++ არქიტექტურა იყო წარმატებული მცდელობა ამ ნაპრალის შესამცირალურ მოდელირებაზე. შემოთავაზებულ ნაშრომში ჩვენ ამ პრობლემას სხვა განზომილებიდან — LISA რეგულარიზაციისა და სემანტიკური ანკორირების მეშვეობით ვუდგებით, რაც მათემატიკურად ათანაბრებს ვიზუალურ ტოპოლოგიას ტექსტურ ორიენტირებთან.

დოსოვიცკის (2021) მიერ შემოთავაზებულმა ViT მოდელმა მოახდინა რევოლუცია კომპიუტერულ ხედვაში გამოსახულების პატჩებად (patches) დაყოფის გზით, რაც მათ NLP ტოკენების მათემატიკურ ანალოგად აქცევს. ეს მიდგომა CXR-ანალიზში კრიტიკულია, რადგან საშუალება იძლევა მოდელირდეს გლობალური დამოკიდებულებები ფილტვის სხვადასხვა ზონებს შორის პირველივე ფენებიდანვე.\footnote{დოსოვიცკი, ა., ბეიერი, ლ., კოლესნიკოვი, ა. და სხვა, „An Image is Worth 16x16 Words: Transformers for Image Recognition at Scale“, International Conference on Learning Representations (ICLR), 2021.}

\subsection{CNN არქიტექტურები და მათი ლიმიტები}
ResNet, DenseNet და EfficientNet ოჯახის მოდელებმა აჩვენეს მაღალი ეფექტუყოფა ლოკალური ტექსტურული ნიშნების ამოცნობაში. თუმცა, კონვოლუციური ქსელების მთავარი ნაკლი ლოკალური რეცეპტული ველია, რაც ართულებს გლობალური ანატომიური კონტექსტის აღქმას მრავალნიშნიან ჭრილში.

ბალტრუშატმა და სხვებმა (2019) Grad-CAM-ის ანალიზით აჩვენეს, რომ ტრადიციული CNN-ები ხშირად უშვებენ კრიტიკულ შეცდომებს, რადგან პათოლოგიის ნაცვლად რეაგირებენ ისეთ ვიზუალურ არტეფაქტებზე, როგორიცაა სამედიცინო მოწყობილობები (მაგ. დრენაჟი პნევმოთორაქსის დროს). ეს კვლევა ხაზს უსვამს მოდელის ინტერპრეტირებადობისა და სემანტიკური რეგულარიზაციის აუცილებლობას.\footnote{ბალტრუშატი, ი. მ., ნიკიში, ჰ., გრასი, მ. და სხვა, „Comparison of Deep Learning Approaches for Multi-Label Chest X-Ray Classification“, Scientific Reports, ტომი 9, ნომერი 6381, 2019.}

\subsection{ტრანსფერული სწავლება და ფუნდამენტური მოდელები (Foundation Models)}
სამედიცინო ანოტაციების სიმცირის გამო, ტრანსფერული სწავლება (Transfer Learning) კვლავ კრიტიკულ როს ასრულებს. 2024-2026 წლების SOTA მიდგომები გადავიდა ImageNet-ის პრიორებიდან სპეციალიზებულ სამედიცინო ფუნდამენტურ მოდელებზე, როგორიცაა Google-ის ELIXR-C.\footnote{ქსუ, ს., იანგი, ლ., კელი, ს. და სხვა, „ELIXR: Towards a General Purpose X-ray Artificial Intelligence System Through Alignment of Large Language Models and Radiology Vision Encoders“, arXiv preprint arXiv:2308.01317, 2024.}

\subsection{Vision Transformers (ViT) და სივრცითი ყურადღება}
Vision Transformer-ებმა (ViT) რევოლუცია მოახდინეს სამედიცინო AI-ში Self-Attention მექანიზმის წყალობით. ისინი საშუალებას იძლევიან მოდელირდეს კავშირი გამოსახულების დაშორებულ რეგიონებს შორის, რაც კრიტიკულია ისეთი დაავადებების დროს, როგორიცაა კარდიომეგალია ან პნევმოთორაქსი, სადაც ანატომიური პროპორციები გადამწყვეტია.\footnote{დოსოვიცკი, ა., ბეიერი, ლ., კოლესნიკოვი, ა. და სხვა, „An Image is Worth 16x16 Words: Transformers for Image Recognition at Scale“, International Conference on Learning Representations (ICLR), 2021.} შემოთავაზებულ მოდელში აღნიშნული თვისება გაძლიერებულია ორგანზომილებიანი როტაციული პოზიციური ემბედინგებით (2D RoPE), რაც ზრდის ლოკალიზაციის მათემატიკურ სიზუსტეს.

\section{ChestMNIST: მრავალნიშნიანი კლასიფიკაციის ბენჩმარკი}
ChestMNIST (MedMNIST v2) წარმოადგენს გულ-მკერდის რენტგენოგრაფიის სტანდარტიზებულ ბენჩმარკს, რომელიც აერთიანებს 14 განსხვავებულ პათოლოგიას. ამოცანა რთულდება კლასთა მკვეთრი დისბალანსით და კო-მორბიდული დაავადებების მაღალი სიხშირით.\footnote{იანგი, ჯ., ში, რ., ვეი, დ. და სხვა, „MedMNIST v2: A Large-Scale Lightweight Benchmark for 2D and 3D Biomedical Image Classification“, Scientific Data, ტომი 10, ნომერი 1, 2023.} ლიტერატურის ანალიზი აჩვენებს, რომ ამ ბაზაზე... 

MedMNIST v2-ის ფარგლებში არსებული ChestMNIST ქვესიმრავლე მოიცავს 112,120 ფრონტალურ რენტგენოგრამას, სადაც თითოეული გამოსახულება ანოტირებულია 14 სხვადასხვა პათოლოგიის მიხედვით. იანგის (2023) მიერ დადგენილი პროტოკოლი უზრუნველყოფს მონაცემთა გაყოფას (Train/Val/Test) პაციენტის დონეზე, რაც მათემატიკურად გამორიცხავს მონაცემთა გაჟონვას (Data Leakage) და კვლევას ხდის მაღალი მაღალი რეპროდუცირებადობის მქონეს. ჩვენს ექსპერიმენტში გამოყენებულია ამ მონაცემების სტანდარტიზებული 10\%-იანი ფუნდამენტური ექსტრაქტი (7,846 სატრენინგო, 1,121 სავალიდაციო და 2,243 სატესტო ნიმუში).

\section{გამოწვევები: გაურკვევლობა და კალიბრაცია}
სამედიცინო გადაწყვეტილების მიღებისას კრიტიკულია, რომ მოდელმა „იცოდეს, რა არ იცის“.
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{Overconfidence}: ტრადიციული Sigmoid-ზე დაფუძნებული მოდელები ხშირად აბრუნებენ მაღალ ალბათობებს არასწორ პროგნოზებზე.
    \item \textbf{Calibration Drift}: მოდელის მიერ ნავარაუდები ალბათობა ხშირად არ ემთხვევა დაავადების რეალურ ემპირიულ სიხშირეს.
    \item \textbf{Epistemic Uncertainty}: გაურკვევლობა, რომელიც გამოწვეულია საწვრთნელი მონაცემების ნაკლებობით ან კოგნიტური დეფიციტით, საჭიროებს დიფერენცირებულ მოდელირებას.\footnote{სენსოი, მ., კაპლანი, ლ. მ., კანდემირი, მ., „Evidential Deep Learning to Quantify Classification Uncertainty“, Advances in Neural Information Processing Systems (NeurIPS), 2018.}
\end{enumerate}

კალიბრაციის პრობლემა განსაკუთრებით გამწვავდა მასიური არქიტექტურების (Model Capacity) და Batch Normalization-ის დანერგვასთან ერთად. გუოს (2017) კვლევამ გამოავლინა დისკონექტი NLL-ის (Negative Log Likelihood) ოპტიმიზაციასა და მოდელის რეალურ პროგნოზულ კალიბრაციას შორის, რამაც საფუძველი ჩაუყარა პოსტ-ჰოკ (post-hoc) კალიბრაციის მეთოდების განვითარებას.\footnote{გუო, ჩ., პლეისი, ჯ., სანი, ი. და სხვა, „On Calibration of Modern Neural Networks“, Proceedings of the 34th International Conference on Machine Learning (ICML), 2017. გვ-გვ. 1321–1330.}

გაურკვევლობის რაოდენობრივი შეფასების ტრადიციული ბაიესური მეთოდები (BNNs) მოითხოვს დიდ გამოთვლით რესურსებს და მონტე-კარლო ინტეგრაციას. სენსოიმ (2018) შემოიტანა რადიკალურად განსხვავებული მიდგომა: ნაცვლად წერტილოვანი შეფასებისა, მოდელი პროგნოზირებს ევიდენციური განაწილების პარამეტრებს. შემოთავაზებულ მოდელში აღნიშნული ჩარჩო რეალიზებულია ბინომიალური საკლასო სუბიექტური ლოგიკის (Subjective Logic) თეორიის საფუძველზე, სადაც თითოეული კლასისთვის ცალ-ცალკე ხდება ეპისტემური გაურკვევლობის (Shannon Entropy-ს საფუძველზე) და ალეატორული გაურკვევლობის ($p \cdot (1-p)$ ბინომიალური ვარიაციის სახით) ზუსტი მათემატიკური გამოთვლა.

\section{ენსამბლირება და ემპირიული ვალიდაცია}
ტრანსფერული სწავლება და პარამეტრულად ეფექტური წვრთნა (Parameter-Efficient Fine-Tuning) სამედიცინო გამოსახულებების ანალიზში იძლევა გენერალიზაციის მაღალ ხარისხს. თუმცა, მონაცემთა ასიმეტრიული განაწილებისა და იშვიათი პათოლოგიების პირობებში, ცალკეული მოდელები ხშირად განიცდიან ტრენინგის არასტაბილურობას. 

ამ პრობლემის გადასაჭრელად, ჩვენს სისტემაში დაინერგა 3-სიდიანი (seeds) დამოუკიდებელი მოდელების წვრთნის ციკლი სტოქასტური წონების გაშუალებით (Stochastic Weight Averaging - SWA). ლოგების ანალიზი აჩვენებს, რომ თითოეული მოდელი სტაბილურად აღწევს ოპტიმალურ კლინიკურ კონვერგენციას (ვალიდაციის პიკური AUC $\approx 0.75-0.76$, Ep 20-23). ენსამბლის დონეზე მიღწეული მაკრო-AUC შეადგენს **0.7889**-ს, რაც მათემატიკურად ამტკიცებს ენსამბლირების ეფექტურობას სამედიცინო გადაწყვეტილების მიღებისას.
\section{კვლევითი ხარვეზი და ნაშრომის წვლილი}
არსებული ლიტერატურის კრიტიკული ანალიზი ცხადყოფს, რომ კვლევების უმეტესობა ფოკუსირებულია მხოლოდ დისკრიმინაციულ სიზუსტეზე (AUC), ხოლო უგულებელყოფილია სამი კომპონენტის ერთობლიობა:
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{Representational Alignment}: გაყინული ენკოდერების ადაპტაცია კლინიკური ონტოლოგიების (RadLex) გამოყენებით;
    \item \textbf{Long-tail Stability}: იშვიათი პათოლოგიების დაცვა ეფექტური ნიმუშების რაოდენობის (ENS, $\beta = 0.9999$) ასიმეტრიული გათვალისწინებით;
    \item \textbf{Trustworthiness Stack}: ერთდროულად მაღალი კალიბრაციის (Mean ECE = 1.17\%), ეპისტემური გაურკვევლობის დეკუპლირებისა და კონფორმალური უსაფრთხოების გარანტიების (CRC) მიღწევ.
\end{enumerate}

წინამდებარე ნაშრომი ავსებს ამ ხარვეზს შემოთავაზებული მოდელის წარმოდგენით, რომელიც აერთიანებს LISA რეგულარიზაციას, PaQ ყურადღების მექანიზმს და მათემატიკურად გარანტირებულ კლინიკურ დაფარვას.

\section{თვითზედამხედველური სწავლება ვიზუალურ რეპრეზენტაციებში}
თვითზედამხედველური სწავლება (Self-Supervised Learning - SSL) კრიტიკულ მნიშვნელობას იძენს სამედიცინო გამოსახულებების ანალიზში, სადაც მაღალ ხარისხის ანოტაციების მოპოვება დაკავშირებულია მაღალ ფინანსურ და დროით ხარჯებთან. DINO/DINOv2 და MAE ტიპის არქიტექტურებმა დაამტკიცეს, რომ დიდი მოცულობის არალეიბლიანი მონაცემებიდან შესაძლებელია ისეთი მახასიათებლების სივრცის (Feature Space) ექსტრაქცია, რომელიც downstream ამოცანებზე მცირე ანოტირებული ნაკრებითაც ეფექტურად მუშაობს. რადიოლოგიაში ამ პარადიგმას ავითარებენ სპეციალიზებული ფუნდამენტური მოდელები (CXR Foundation Models), როგორიცაა ELIXR-C, რომლებიც უშუალოდ გულ-მკერდის რენტგენოგრამების გლობალურ კონტექსტსა და ლოკალურ ანომალიებზეა ოპტიმიზებული.

წინამდებარე ნაშრომში გამოყენებულია სწორედ აღნიშნული მიდგომა: მოდელი აღარ სწავლობს საწყისი პიქსელებიდან (Raw Pixels), არამედ ოპერირებს წინასწარ ექსტრაქტირებულ $8 \times 8$ სივრცით მახასიათებლებზე (1376 განზომილება). ეს მიდგომა რადიკალურად ამცირებს გამოთვლით რესურსებს და საშუალებას იძლევა კვლევის ფოკუსი გადავიდეს ლეიბლების სემანტიკაზე, გაურკვევლობის მოდელირებასა და კონფორმალურ უსაფრთხოზე (Conformal Safety).

\section{ტრანსფერული სწავლება სამედიცინო გამოსახულებებში}
ტრანსფერული სწავლების (Transfer Learning) მთავარი უპირატესობა მონაცემთა ეკონომია და კონვერგენციის სტაბილურობაა. სამედიცინო მონაცემთა ნაკრებები ხშირად მცირეა იმ მასშტაბთან შედარებით, რაც თანამედროვე ღრმა მოდელებს სჭირდებათ. ImageNet-ზე გაწვრთნილი ტრადიციული მოდელები (ResNet/EfficientNet) წლების განმავლობაში წარმოადგენდა სტანდარტს, თუმცა სამედიცინო დომენური ცვლა (Domain Shift) ხშირად ზღუდავდა მათ ეფექტურობას. სამედიცინო ფუნდამენტური რეპრეზენტაციები ამცირებს ამ პრობლემას, რადგან ვიზუალური ენკოდერი უკვე ფლობს ინფორმაციას რადიოლოგიური ინტენსივობების, ანატომიური პოზიციებისა და CXR-სპეციფიკური ტექსტურების შესახებ.

შემოთავაზებულ მოდელში ELIXR-C ბექბოუნი (Backbone) გამოიყენება გაყინულ (Frozen) რეჟიმში. ეს არის გააზრებული მეთოდოლოგიური არჩევანი: მახასიათებლების ფიქსაცია ზრდის ექსპერიმენტების რეპროდუცირებადობას, ინარჩუნებს ვიზუალურ სტაბილურობას და საშუალებას იძლევა იზოლირებულად შეფასდეს კლასიფიკატორის თავის წვლილი რთული პათოლოგიების ამოცნობაში.

\section{ღრმა ენსამბლები და აუგმენტაციები}
ერთიანი დეტერმინისტული მოდელი ხშირად ავლენს ზე-თავდაჯერებულობას (Overconfidence), რაც კლინიკური რისკის წყაროა. Deep Ensemble მეთოდოლოგია ამცირებს ამ პრობლემას რამდენიმე დამოუკიდებელი მოდელის კომბინაციით. თუ სხვადასხვა მოდელები ერთსა და იმავე გამოსახულებაზე განსხვავებულ ალბათობებს აბრუნებენ, ეს მიუთითებს ეპისტემურ გაურკვევლობაზე (Epistemic Uncertainty) — ანუ მოდელის ცოდნის დეფიციტზე მოცემულ პატერნთან მიმართებაში.

ჩვენს მეთოდოლოგიაში ენსამბლი შედგება დამოუკიდებელი მოდელებისგან, ხოლო ინფერენსის ეტაპზე გამოყენებულია MC-Dropout-ის 10 ციკლი. საბოლოო პროგნოზი წარმოადგენს ყველა ციკლისა და ყველა მოდელის საშუალო მაჩვენებელს ლოგიტების სივრცეში, რაც ინარჩუნებს წრფივ კავშირებს. ეს მიდგომა უზრუნველყოფს პრაქტიკულ ბაიესურ მიახლოებას (Bayesian Approximation), რაც კრიტიკულად მნიშვნელოვანია კლინიკური გადაწყვეტილების მხარდასაჭერად.


\subsection{კალიბრაცია და კონფორმალური პროგნოზირება}
კალიბრაცია განსაზღვრავს მოდელის პროგნოზირებული ალბათობის შესაბამისობას რეალურ ემპირიულ სიხშირესთან. თუ მოდელი პათოლოგიის არსებობას 80\% ალბათობით აფასებს, ჭეშმარიტად დადებითი შემთხვევების წილი ამ ჯგუფში ზუსტად 80\% უნდა იყოს. კალიბრაციის ხარისხი იზომება ECE (Expected Calibration Error) მეტრიკით, რომელიც bin-ების მიხედვით აფასებს ამ ცდომილებას. 

ჩვენს კვლევაში, კლასობრივად ასიმეტრიული იზოტონური რეგრესიის (AIR) გამოყენებამ მოახდინა კალიბრაციის დრამატული გაუმჯობესება: არაკალიბრირებული ECE (**0.3034**) შემცირდა სტატისტიკურად თითქმის ნულოვან ვალიდაციურ მაჩვენებლამდე (**0.0003**), ხოლო სატესტო ნაკრებზე მიღწეულ იქნა საბოლოო საშუალო ECE: **0.0117** ($1.17\%$).

კონფორმალური პროგნოზირება (Conformal Prediction) სისტემას ამატებს მათემატიკურ უსაფრთხოების ფენას, რომელიც დამოუკიდებელია მონაცემთა განაწილებისგან (Distribution-free). მრავალლეიბლიან ამოცანაში იგი აგენერირებს პროგნოზების სიმრავლეს (Prediction Set). ფართო სიმრავლე მიუთითებს მოდელის სიფრთხილესა და მაღალ გაურკვევლობაზე. შემოთავაზებული მოდელის გაურკვევლობით მართული კონფორმალური ჩარჩო (CRC) ვალიდაციის სეტზე არჩევს ოპტიმალურ კლასობრივ მულტიპლიკატორებს. 

საინტერესოა, რომ ეპისტემურ გაურკვევლობასა და კონფორმული სიმრავლის ზომას შორის ემპირიულმა კორელაციამ აჩვენა პრაქტიკულად ნულოვანი მნიშვნელობა (Spearman $\rho = -0.0036$, $p = 0.865$). ეს მათემატიკურად ადასტურებს, რომ კონფორმალური რისკის კონტროლი და ეპისტემური ენტროპია ასახავენ განსხვავებულ, ორთოგონალურ ინფორმაციულ საზღვრებს. გარდა ამისა, სელექციური უარყოფის (Abstention) ანალიზმა აჩვენა, რომ სიზუსტე-უარყოფის მრუდი აღწევს მაქსიმალურ პიკს **20\%-იანი რეფერალისას (Retained AUC = 0.7927)**, რის შემდეგაც იწყება "სელექციური პროგნოზირების პარადოქსი" კლასობრივი დისბალანსის გამო. ნულოვანი ზომის სიმრავლეები კი გვხვდება შემთხვევების მხოლოდ 4.0\%-ში (89 პაციენტი) და ექსკლუზიურად ვრცელდება უარყოფილ (მაღალი რისკის მქონე) შემთხვევებზე, რაც უსაფრთხოების გამორჩეული გარანტიაა.

კონფორმალური პროგნოზირების ჩარჩო 2024-2026 წლებში მნიშვნელოვნად განვითარდა. თუ ადრეული სამუშაოები ფოკუსირებული იყო მონოტონურ დანაკარგებზე, ანგელოპულოსის\footnote{ანგელოპულოსი, ა. ნ., ბეიტსი, ს., ფიში, ა. და სხვა, „Conformal Risk Control“, arXiv preprint arXiv:2208.02814, 2022.} უახლესმა კვლევამ მათემატიკურად დაამტკიცა, რომ რისკის კონტროლი შესაძლებელია არამონოტონური ფუნქციებისთვისაც, რაც კრიტიკულია მრავალლეიბლიანი სამედიცინო დიაგნოსტიკისთვის.

\subsection{მრავალნიშნიანი სწავლება და კლასთა დისბალანსი}
გულ-მკერდის რენტგენოგრამებზე პათოლოგიები ხშირად თანაარსებობენ, რაც მოითხოვს მრავალნიშნიანი სწავლების (Multilabel Learning) პარადიგმას. ამ დროს თითოეული ლეიბლი განიხილება როგორც დამოუკიდებელი ბინარული სამიზნე, თუმცა მათი კლინიკური ურთიერთკავშირი (Co-morbidity) არქიტექტურულად უნდა იყოს გათვალისწინებული.

კლასთა დისბალანსი განსაკუთრებით მწვავეა იშვიათ პათოლოგიებზე, სადაც მოდელები ხშირად მიდრეკილნი არიან False Negative შეცდომებისკენ. შემოთავაზებული მოდელი ამ პრობლემას წყვეტს ორი სინერგიული გზით: (1) კლასობრივად ბალანსირებული ასიმეტრიული დანაკარგის ფუნქციით (CB-ASL), რომელიც იყენებს ეფექტური ნიმუშების რაოდენობის (Effective Number of Samples - ENS) მიხედვით კალიბრირებულ კლასობრივ წონებს და (2) ვექტორიზებული კლასობრივი გადაწყვეტილების ზღვრების ოპტიმიზაციით (F1-Maximized Threshold Optimization), რაც უზრუნველყოფს მაკრო-F1 მეტრიკის მაქსიმიზაციას. გარდა ამისა, PaQ (Pathology-as-Query) მექანიზმი საშუალებას აძლევს მოდელს, RadLex ონტოლოგიაზე დაყრდნობით, იშვიათი პათოლოგიების ნიშნები სემანტიკურად დაკავშირებულ უფრო ხშირ დაავადებებში „ეძებოს“.

\chapter{მეთოდოლოგია და არქიტექტურა}

\section{მონაცემთა ბაზა და გაყოფის სტრატეგია}
კვლევის ემპირიულ საფუძველს წარმოადგენს MedMNIST v2-ის ChestMNIST მონაცემთა ბაზა, რომელიც მოიცავს გულ-მკერდის რენტგენოგრამების 14-ლეიბლიან მრავალნიშნიან კლასიფიკაციას. მონაცემთა ჩატვირთვა ხორციელდება \texttt{medmnist.ChestMNIST} ინტერფეისით, 224x224 პიქსელური გარჩევადობით. რეპროდუცირებადობის უზრუნველსაყოფად, ნაშრომში დაცულია MedMNIST-ის ოფიციალური split-პროტოკოლი.

მონაცემთა დამუშავებისას დაცულია MedMNIST v2-ის ოფიციალური რეკომენდაციები. მიუხედავად იმისა, რომ ბაზის ორიგინალი ვერსია 28x28 რეზოლუციაზეა ორიენტირებული, ჩვენს მოდელში გამოყენებულია ChestMNIST-ის გაფართოებული 224x224 ვერსია, რაც ავტორების\footnote{იანგი, ჯ., ში, რ., ვეი, დ. და სხვა, „MedMNIST v2: A Large-Scale Lightweight Benchmark for 2D and 3D Biomedical Image Classification“, Scientific Data, ტომი 10, ნომერი 1, 2023.} მოსაზრებით ინარჩუნებს დიაგნოსტიკურად რელელვანტურ ტექსტურულ ნიშნებს უფრო მაღალი დისკრიმინაციული უნარისთვის. კვლევაში გამოყენებულია ამ მონაცემების 10\%-იანი ფუნდამენტური დეცილური ექსტრაქტი, რაც ჯამში შეადგენს 7,846 სატრენინგო, 1,121 სავალიდაციო და 2,243 სატესტო ნიმუშს.

ბალტრუშატის (2019) ექსპერიმენტულმა შედეგებმა დაადასტურა, რომ არავიზუალური მეტამონაცემების ინტეგრირება სისტემურად ზრდის დიაგნოსტიკურ სიზუსტეს. აღნიშნულ კვლევაზე დაყრდნობით, შემოთავაზებული მოდელის არქიტექტურაში გათვალისწინებულია პაციენტის დემოგრაფიული მონაცემებისა და სკანირების პოზიციის გავლენა საბოლოო ვერდიქტზე.\footnote{ბალტრუშატი, ი. მ., ნიკიში, ჰ., გრასი, მ. და სხვა, „Comparison of Deep Learning Approaches for Multi-Label Chest X-Ray Classification“, Scientific Reports, ტომი 9, ნომერი 6381, 2019.}

მონაცემთა ვალიდურობისთვის დაინერგა \textbf{Forensic Cross-Split Hashing Audit} მექანიზმი. \texttt{data.py} მოდული თითოეული გამოსახულებისთვის აგენერირებს ბიტ-ზუსტ Blake2b კრიპტოგრაფიულ ჰეშ-ფინგერპრინტებს. ეს საშუალებას გვაძლევს გამოვრიცხოთ მონაცემთა დაბინძურება (Contamination) და გაჟონვა სატესტო სიმრავლეებში, რაც ხშირად ხდება სამედიცინო ბაზებში ერთი და იმავე პაციენტის სხვადასხვა სკანის სხვადასხვა split-ში მოხვედრისას. ჩატარებულმა ფორენზიკულმა აუდიტმა (Run ID: 0b288e872347) დაადასტურა, რომ კროს-სპლიტ გაჟონვის მაჩვენებელი არის ზუსტად 0 (Strictly Separated), რაც გარანტიას იძლევა მიღებული შედეგების მათემატიკურ ვალიდურობაზე. დამატებით, \texttt{audit.py} ახორციელებს კლინიკური ლოგიკის შემოწმებას, მათ შორის პათოლოგიათა თანაარსებობის (Co-occurrence) ანალიზს და სიგნალ-ხმაურის (SNR) განაწილების მონიტორინგს.

\section{ფუნდამენტური მახასიათებლების ექსტრაქცია}
შემოთავაზებული მოდელის არქიტექტურა End-to-End პიქსელური სწავლების ნაცვლად ეყრდნობა \textbf{Frozen-Foundation} პარადიგმას. მონაცემთა წინასწარი დამუშავებისას, \texttt{extraction\_pipeline.py} ChestMNIST-ის გამოსახულებებს აწვდის Google-ის ELIXR-C (CXR-Foundation v2) მოდელს. 

ეს მოდელი თითოეული სურათიდან აგენერირებს $8 \times 8$ სივრცითი განზომილების მქონე feature map-ს, სადაც თითოეული წერტილი (Patch) წარმოდგენილია 1376-განზომილებიანი ვექტორით. ექსპერიმენტული ციკლის ეფექტურობისთვის გამოყენებულია მონაცემთა 10\%-იანი დეცილური ქვესიმრავლე (\texttt{FRACTION=0.10}), რაც ინარჩუნებს სტატისტიკურ სარწმუნოობას და ამცირებს გამოთვლით ხარჯებს. ექსპერიმენტებისთვის მონაცემთა 10\%-იანი ქვესიმრავლის (decile) გამოყენება ეყრდნობა ELIXR-ის (2024) კვლევის შედეგებს, რომლის მიხედვითაც, ფუნდამენტური ემბედინგების გამოყენება საშუალებას იძლევა მიღწეულ იქნას SOTA მაჩვენებლები ორი რიგით ნაკლები მონაცემების პირობებში, ვიდრე ეს სტანდარტული ზედამხედველობითი მოდელების შემთხვევაშია შესაძლებელი.\footnote{ქსუ, ს., იანგი, ლ., კელი, ს. და სხვა, „ELIXR: Towards a General Purpose X-ray Artificial Intelligence System Through Alignment of Large Language Models and Radiology Vision Encoders“, arXiv preprint arXiv:2308.01317, 2024.}

\section{შემოთავაზებული მოდელის არქიტექტურა}
შემოთავაზებული მოდელის არქიტექტურის დიზაინი ეფუძნება პრინციპს, რომ სამედიცინო დიაგნოსტიკა მოითხოვს არა მხოლოდ პიქსელების ანალიზს, არამედ კლინიკური ცოდნის ინტეგრაციას. ალიფერისი და სიმონის (2024) კრიტიკულად აღნიშნავენ, რომ თანამედროვე ღრმა ნეირონული ქსელები, მიუხედავეად მათი შრეების სიმრავლისა, ხშირად „ონტოლოგიურად ზედაპირულია“ (\textit{ontologically shallow}), რადგან მათ არ გააჩნიათ სამედიცინო დომენისა და მისი ობიექტების მდიდარი სემანტიკური რეპრეზენტაცია.\footnote{სიმონი, ჯ. ჯ., ალიფერისი, ქ., რედ., „Artificial Intelligence and Machine Learning in Health Care and Medical Sciences: Best Practices and Pitfalls“, Springer, Cham, 2024. გვ-გვ. 15–16.} 

ამ ფუნდამენტური შეზღუდვის დასაძლევად, ჩვენს მოდელში დანერგილია ცოდნით განპირობებული ორი სინერგიული მექანიზმი: 
\begin{enumerate}
    \item \textbf{Pathology-as-Query (PaQ)} სივრცითი ყურადღების მექანიზმი, რომელიც ობიექტზე ორიენტირებულ ინფერენსს ახორციელებს;
    \item \textbf{LISA (Language-Invariant Semantic Anchoring)} რეგულარიზაცია, რომელიც უზრუნველყოფს მოდელის სემანტიკურ ანკორირებას RadLex-ის პროფესიულ ონტოლოგიაში;
    \item \textbf{2D Rotary Position Embeddings (2D RoPE)} ორგანზომილებიანი სივრცითი ბრუნვის პრინციპი, რომელიც ჩაშენებულია უშუალოდ Cross-Attention მოდულში.
\end{enumerate}

გამოსახულების სემანტიკური კორექციის იდეა CXR-ში ადრეც იყო შესწავლილი, მაგალითად Dai et al. (2018) იყენებდნენ ადვერსალურ ქსელებს (SCAN) Lung-fields-ის სტრუქტურული სიზუსტისთვის. ჩვენი Pathology-as-Query (PaQ) მექანიზმი ავითარებს ამ ფილოსოფიას და ადვერსალური ჯარიმების ნაცვლად იყენებს გრაფზე დაფუძნებულ კლინიკურ მიზეზობრიობას ანატომიური თანმიმდევრულობის უზრუნველსაყოფად.

აღნიშნული მიდგომა საშუალებას აძლევს სისტემას, გასცდეს პატერნების მარტივ ამოცნობას და დაეყრდნოს კლინიკურად ვალიდურ სემანტიკურ ტოპოლოგიას. Baltruschat et al. (2019) მიერ ჩატარებულმა Grad-CAM ანალიზმა გამოავლინა კრიტიკული შეცდომა: მოდელმა ისწავლა პნევმოთორაქსის ამოცნობა სამედიცინო დრენაჟების მიხედვით (Shortcut Learning). ჩვენს არქიტექტურაში LISA რეგულარიზაცია BioViL-T ემბედინგებით უზრუნველყოფს მოდელის სემანტიკურ ანკორირებას, რაც აიძულებს მას ფოკუსირება მოახდინოს ანატომიურ ნიშნებზე და არა ვიზუალურ არტეფაქტებზე.\footnote{ბალტრუშატი, ი. მ., ნიკიში, ჰ., გრასი, მ. და სხვა, „Comparison of Deep Learning Approaches for Multi-Label Chest X-Ray Classification“, Scientific Reports, ტომი 9, ნომერი 6381, 2019.}

\subsection{1. განზომილების შემცირება და სტაბილიზაცია}
ELIXR-C-ის მაღალგანზომილებიანი (1376-D) ვექტორები გადიან პროექციას 384-განზომილებიან სამუშაო სივრცეში. ეს პროცესი ხორციელდება ორეტაპიანი ხაზოვანი გარდაქმნით Layer-Norm და GELU აქტივაციის ფენების გამოყენებით, რაც უზრუნველყოფს ნიშნების სტაბილურობას შემდგომი ყურადღების (Attention) მექანიზმისთვის:
\begin{equation}
h = \text{LayerNorm}(\text{GELU}(W_{\text{proj2}} \cdot \text{Dropout}(\text{GELU}(\text{LayerNorm}(W_{\text{proj1}} \cdot x)))))
\end{equation}

\subsection{2. ორგანზომილებიანი როტაციული პოზიციური ემბედინგი (2D RoPE)}
ნეირონული ქსელისთვის ანატომიური კონტექსტის მისანიჭებლად და გეომეტრიული მთლიანობის შესანარჩუნებლად, მოდელი უარს ამბობს ტრადიციულ ადიტიურ პოზიციურ კოდირებაზე და იყენებს მოწინავე **ორგანზომილებიან როტაციულ პოზიციურ ემბედინგს (2D RoPE)**. ეს მექანიზმი ინტეგრირებულია უშუალოდ \texttt{PathologyCrossAttention} შრის შიგნით, კროს-ყურადღების გამოთვლამდე.

მოდელი $384$-განზომილებიან სივრცეს ყოფს ორ დამოუკიდებელ ქვე-სივრცედ $d_{\text{half}} = 192$ განზომილებით თითოეული კოორდინატისთვის ($x$ და $y$). სიხშირეების ვექტორი $\theta$ იანგარიშება შემდეგნაირად:
\begin{equation}
\theta_i = \frac{1}{10000^{2i / d_{\text{half}}}}, \quad i \in \left[0, \dots, \frac{d_{\text{half}}}{2}-1\right]
\end{equation}

თითოეული $8 \times 8$ პატჩისთვის, კოორდინატები $x, y \in [0, \dots, 7]$ პროეცირდება კუთხურ ფაზებად:
\begin{equation}
\Theta_x = x \otimes \theta_i, \quad \Theta_y = y \otimes \theta_i
\end{equation}

მიღებული ფაზები გაერთიანდება $192$-განზომილებიან ერთიან ვექტორად $\Theta = [\Theta_x, \Theta_y]$, ხოლო როტაციული ტრანსფორმაცია ვიზუალურ $h$ ვექტორზე სრულდება მულტიპლიკაციური ბრუნვით:
\begin{equation}
R(h, \Theta) = h \odot \cos(\Theta) + \text{rotate\_half}(h) \odot \sin(\Theta)
\end{equation}
სადაც $\text{rotate\_half}(h) = [-h_{d/2:d}, h_{0:d/2}]$. ეს მათემატიკური მიდგომა უზრუნველყოფს, რომ ყურადღების წონები ექსპონენციალურად მცირდებოდეს ანატომიურ პატჩებს შორის მანძილის ზრდასთან ერთად, რაც გადამწყვეტია კარდიომეგალიისა და პნევმოთორაქსის ზუსტი სივრცითი დიაგნოსტიკისთვის.\footnote{დოსოვიცკი, ა., ბეიერი, ლ., კოლესნიკოვი, ა. და სხვა, „An Image is Worth 16x16 Words: Transformers for Image Recognition at Scale“, International Conference on Learning Representations (ICLR), 2021.}

\subsection{3. Pathology-as-Query (PaQ) Routing}
არქიტექტურის ცენტრალური ინოვაციაა \textbf{PaQ Cross-Attention} მექანიზმი. თითოეული 14 პათოლოგია წარმოდგენილია როგორც RadLex-ზე დაფუძნებული ტექსტური შეკითხვა (Query). 

ჩვენს მიერ დანერგილი PaQ მექანიზმი იზიარებს Wang et al. (2026) მიერ შემოთავაზებულ Attribute-Centric ყურადღების ლოგიკას. აღნიშნული არქიტექტურა არღვევს „vision-to-text“ ინფორმაციულ ბოთლის ყელს (bottleneck) და ვიზუალურ ენკოდერს პირდაპირ აწვდის კლინიკურ ცოდნას სპეციალიზებული ტოკენების (ჩვენს შემთხვევაში RadLex ემბედინგების) მეშვეობით.\footnote{ვანგი, ჰ., ქსუ, ქს., ვანგი, ლ. და სხვა, „AttTok: Marrying Attribute Tokens with Generative Pre-trained Vision-Language Models towards Medical Image Understanding“, International Conference on Learning Representations (ICLR), 2026.} \texttt{ensure\_radlex\_embeddings} ფუნქცია იყენებს ბიოსამედიცინო ენობრივ მოდელს — \textbf{BioViL-T}, რათა კლინიკური ტერმინები გარდაქმნას 768-განზომილებიან ემბედინგებად.

მიუხედავად იმისა, რომ ELIXR იყენებს Q-Former-ს ზოგადი ვიზუალური ტოკენების ექსტრაქციისთვის, შემოთავაზებული მოდელი ნერგავს Pathology-as-Query (PaQ) მექანიზმს. ეს მიდგომა სემანტიკურად უფრო ღრმაა: ჩვენ ვიყენებთ ELIXR-ის (2024) მიერ ვალიდირებულ კლინიკურ პრომპტებს როგორც პირდაპირ შეკითხვებს (Queries) ყურადღების შრისთვის.\footnote{ქსუ, ს., იანგი, ლ., კელი, ს. და სხვა, „ELIXR: Towards a General Purpose X-ray Artificial Intelligence System Through Alignment of Large Language Models and Radiology Vision Encoders“, arXiv preprint arXiv:2308.01317, 2024.} ეს მოდელს საშუალებას აძლევს, მოახდინოს მტკიცებულებების უფრო ზუსტი სივრცითი ატრიბუცია.

ინფერენსის დროს, თითოეული დაავადების ვექტორი (Query) სკანირებს 2D RoPE-ით როტირებულ $8 \times 8$ სივრცით პატჩებს (Key/Value), რათა მოახდინოს პათოლოგიური მტკიცებულებების სივრცითი ლოკალიზაცია. შემოთავაზებულ მოდელში დანერგილი Pathology-as-Query (PaQ) მექანიზმი ტექნიკურ შესაბამისობაშია DCG-Net-ის მიერ შემოთავაზებულ ბიდირექციულ მულტიმოდალურ ყურადღებასთან (Bidirectional Multimodal Alignment). DCG-Net-ის მსგავსად, ჩვენი მოდელი ცვლის სტანდარტულ კოსინუსურ მსგავსებას (Cosine Similarity) დინამიური Cross-Attention-ით, რაც საშუალებას აძლევს ტექსტურ პროტოტიპებს (RadLex) მოახდინონ ვიზუალური მტკიცებულებების ზუსტი სივრცითი ატრიბუცია.

\subsection{4. Language-Invariant Semantic Anchoring (LISA)}
წარმოდგენების დრიფტის (Representational Drift) აღმოსაფხვრელად, დაინერგა LISA რეგულარიზაცია. LISA აიძულებს მოდელს, რომ ტრენინგის განმავლობაში ნასწავლი ვიზუალური შეკითხვები (Queries) არ დაშორდეს თავდაპირველ RadLex სემანტიკურ ტოპოლოგიას. ეს მიიღწევა კოსინუსური მსგავსების მატრიცების მუდმივი შედარებით KL-Divergence დანაკარგის ფუნქციის მეშვეობით, სადაც შედარებულია მიმდინარე ტოპოლოგია ($Q_{\text{current}}$) და BioViL-T ბაზისური პროექცია ($P_{\text{target}}$):
\begin{equation}
\mathcal{L}_{\text{LISA}} = \text{KL}(Q_{\text{current}} \,||\, P_{\text{target}})
\end{equation}

LISA რეგულარიზაციის კონცეპტი ეხმიანება AttTok-ის (2026) პრინციპს, სადაც კლინიკური ატრიბუტები გამოიყენება როგორც Anchor Points (საყრდენი წერტილები). ეს უზრუნველყოფს ვიზუალური და ტექსტური სივრცეების ბი-დირექციულ გასწორებას (Alignment) და ხელს უშლის რეპრეზენტაციულ დრიფტს, რაც Wang et al.-ის მიხედვით, კრიტიკულია ზუსტი დიაგნოსტიკისთვის.\footnote{ვანგი, ჰ., ქსუ, ქს., ვანგი, ლ. და სხვა, „AttTok: Marrying Attribute Tokens with Generative Pre-trained Vision-Language Models towards Medical Image Understanding“, International Conference on Learning Representations (ICLR), 2026.} ეს უზრუნველყოფს, რომ მოდელმა შეინარჩუნოს კლინიკური ლოგიკა მონაცემთა სიმცირის პირობებშიც.

\subsection{5. Class-Balanced Asymmetric Loss (CB-ASL)}
მოდელის სწავლება ხორციელდება სპეციალიზებული კლასობრივად ბალანსირებული ასიმეტრიული დანაკარგის ფუნქციით (**CB-ASL**). ტრადიციული BCE-სგან განსხვავებით, ეს ფუნქცია ახდენს ნეგატიური კლასების ფოკუსირებულ კლებას ($\gamma_{\text{neg}} = 4.0$, კლიპინგი $0.05$) და იყენებს ეფექტური ნიმუშების რაოდენობის (Effective Number of Samples - ENS) მიხედვით ნორმალიზებულ კლასობრივ წონებს $w_c$:
\begin{equation}
E_c = \frac{1 - \beta^{N_c}}{1 - \beta}, \quad w_c = \frac{1}{E_c}
\end{equation}
სადაც $N_c$ არის მოცემული კლასის პოზიტიური ნიმუშების რაოდენობა ტრენინგზე, ხოლო $\beta = 0.9999$. ჯამური საწვრთნელი დანაკარგის ფუნქცია აერთიანებს კლასიფიკაციისა და სემანტიკური ანკორირების (LISA) ამოცანებს:
\begin{equation}
\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{CB-ASL}}(z_v, y; w_c) + \lambda \mathcal{L}_{\text{LISA}}
\end{equation}
სადაც $\lambda = 0.5$. ტრენინგის შემდგომ, ბინომიალური სუბიექტური ლოგიკის საფუძველზე, მოდელის გამოსავალი ლოგიტებიდან ხდება ეპისტემური გაურკვევლობის (Shannon Entropy-ს საფუძველზე) და ალეატორული გაურკვევლობის (ბინომიალური ვარიაციის $p(1-p)$ მეშვეობით) ზუსტი მათემატიკური გამოთვლა, რაც უზრუნველყოფს უსაფრთხოების დამატებით გარანტიებს.

\subsection{6. Graph-Guided Pathology-as-Query (PaQ) Attention}
ტრადიციული მრავალლეიბლიანი მოდელები პათოლოგიებს განიხილავენ როგორც იზოლირებულ ერთეულებს, რაც ეწინააღმდეგება კლინიკური რეალობას, სადაც დაავადებები ხშირად კორელაციაშია (მაგ. Effusion და Consolidation). შემოთავაზებულ მოდელში პათოლოგიური დამოკიდებულებები მოდელირებულია დინამიური ცოდნის გრაფის საშუალებით. ჩვენი გრაფული მატრიცის აგებისას ვითვალისწინებთ DCG-Net (2026)-ის მიერ წარმოდგენილ პრინციპს, სადაც გრაფი ინიციალიზდება სტატისტიკური პრიორებით. თუმცა, შემოთავაზებული მოდელი უფრო შორს მიდის და ამ სტატისტიკურ კავშირებს აძლიერებს LISA რეგულარიზაციით, რაც უზრუნველყოფს სემანტიკურ სტაბილურობას რადიოლოგიურ ონტოლოგიებთან მიმართებაში. 

\texttt{select\_adjacency\_threshold} ფუნქცია იყენებს Elbow-Method-ს, რათა იპოვოს ოპტიმალური ზღვარი პათოლოგიათა თანაარსებობის (Co-occurrence) მატრიცისთვის (ამ ექსპერიმენტში ოპტიმალურად შემოთავაზებულ იქნა $0.050$), ხოლო \texttt{build\_hybrid\_clinical\_adjacency} ქმნის ნორმალიზებულ ლაპლასიანურ მატრიცას. ამის შემდეგ, \texttt{PathologyCrossAttention} ახორციელებს სემანტიკურ მარშრუტიზაციას სამ ეტაპად:
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{Graph Message Passing}: RadLex-ის საწყისი შეკითხვები (Queries) ცვლიან ინფორმაციას კლინიკური გრაფის გასწვრივ. ეს უზრუნველყოფს იშვიათი კლასების (მაგ. Hernia) რეპრეზენტაციის სტაბილიზაციას უფრო ხშირი, სემანტიკურად ახლო კლასების ხარჯზე;
    \item \textbf{Spatial Cross-Attention}: თითოეული გრაფ-ორიენტირებული Query ასკანირებს 2D RoPE-ით როტირებულ ვიზუალურ პატჩებს, რათა მოახდინოს დაავადების ლოკალიზაცია;
    \item \textbf{Cosine Scoring}: საბოლოო ლოგიტი ($z$) მიიღება ნორმალიზებულ ვიზუალურ ნიშნებსა და Query-ს შორის კოსინუსური მსგავსებით, რაც გამრავლებულია მოდელის შიგნით არსებულ სწავლებად მასშტაბირების კოეფიციენტზე (\texttt{logit\_scale}).
\end{enumerate}

\subsection{7. F1-Maximized Decision Threshold Optimization}
კლასთა ექსტრემალური დისბალანსის პირობებში, ფიქსირებული ალბათური ზღვარი (მაგ. $0.50$) იწვევს იშვიათი პათოლოგიების სრულ უგულებელყოფას. ამ პრობლემის გადასაჭრელად, შემოთავაზებული მოდელი უარს ამბობს სტატიკურ ზღურბლებზე და იყენებს **ვექტორიზებულ გადაწყვეტილების ზღურბლების ოპტიმიზატორს**.

ვალიდაციის ეტაპზე, თითოეული კლასისთვის $k$ ცალ-ცალკე, მოდელი გადის ალბათობების სრულ დიაპაზონს (150 ბიჯი) და ირჩევს ზღურბლს $T_k$, რომელიც მაქსიმალურად ზრდის კლასობრივ F1-მეტრიკას. თქვენს ექსპერიმენტში (Run ID: 0b288e872347) ოპტიმიზებულმა ზღურბლებმა აჩვენა კლინიკური კანონზომიერება: მაღალპრევალენტური პათოლოგიებისთვის (მაგ. Effusion) ზღვარმა შეადგინა $0.5402$, ხოლო უკიდურესად იშვიათი პათოლოგიებისთვის (მაგ. Hernia — თიაქარი) ზღვარი ავტომატურად დაიწია $0.1234$-მდე. ეს მათემატიკურად უზრუნველყოფს იშვიათი კლინიკური ანომალიების მაღალი მგრძნობელობით (Sensitivity) დაფიქსირებას.

\section{საწვრთნელი პროცესი და ოპტიმიზაციის სტრატეგია}
საწვრთნელი პროცედურა ოპტიმიზებულია \texttt{train.py}-ს მეშვეობით, რომელიც ითვალისწინებს 35 ეპოქას AdamW ოპტიმიზატორით (Weight Decay = 0.05, Base LR = 2e-4). საწვრთნელი პროცესის სტაბილურობისთვის დანერგილია 200-სტეპიანი წრფივი warmup და შემდგომი Cosine Annealing განრიგი.

\subsection{Class-Balanced Asymmetric Loss (CB-ASL)}
მოდელის სიზუსტისა და კლასობრივი დაცულობის უზრუნველსაყოფად, ტრადიციული BCE ჩანაცვლებულია კლასობრივად ბალანსირებული ასიმეტრიული დანაკარგის ფუნქციით (**CB-ASL**). დანაკარგის ფუნქცია ახდენს ნეგატიური კლასების ფოკუსირებულ კლებას ($\gamma_{\text{neg}} = 4.0$, $\gamma_{\text{pos}} = 1.0$, კლიპინგი $0.05$) და იყენებს ENS კლასობრივ წონებს $w_c$:
\begin{equation}
\mathcal{L}_{\text{CB-ASL}} = - \frac{1}{N} \sum_{n=1}^N \sum_{c=1}^C w_c \left[ y_{n,c} (1 - p_{n,c})^{\gamma_{\text{pos}}} \log(p_{n,c}) + (1 - y_{n,c}) (1 - p'_{n,c})^{\gamma_{\text{neg}}} \log(1 - p'_{n,c}) \right]
\end{equation}
სადაც $p_{n,c} = \sigma(z_{n,c})$ არის მოდელის მიერ ნავარაუდები ალბათობა, ხოლო $p'_{n,c} = \min(1 - p_{n,c} + \text{clip}, 1.0)$ წარმოადგენს კლიპირებულ ალბათობას ნეგატიური კლასების ასიმეტრიული მოდელირებისთვის. ეს მათემატიკურად სჯის მოდელის მიერ გამოხატულ უსაფუძვლო თავდაჯერებულობას, რაც პირდაპირ ამცირებს კალიბრაციის შეცდომას (ECE).

\subsection{LISA: Semantic Anchoring დანაკარგი}
ტრენინგის დროს წარმოდგენების დრიფტის (Representational Drift) თავიდან ასაცილებლად, გამოიყენება \textbf{LISA (Language-Invariant Semantic Anchoring)} დანაკარგი. LISA იყენებს საწყისი RadLex ემბედინგების კოსინუსურ ტოპოლოგიას როგორც მათემატიკურ ღუზას. ტრენინგის პროცესში ნებისმიერი ცვლილება Queries-ს შორის მსგავსების მატრიცაში, რომელიც ეწინააღმდეგება ორიგინალ RadLex სემანტიკას, ჯარიმდება KL-Divergence დანაკარგით (წონით 0.5):
\begin{equation}
\mathcal{L}_{\text{LISA}} = D_{\text{KL}}(P_{\text{current}} \,||\, P_{\text{target}})
\end{equation}
სადაც $P_{\text{current}} = \text{softmax}(Q_{\text{current}} / \tau)$ წარმოადგენს ნასწავლი მოთხოვნების სემანტიკურ მსგავსებას, $P_{\text{target}} = \text{softmax}(Q_{\text{RadLex}} / \tau)$ — ორიგინალ RadLex ონტოლოგიურ მსგავსებას, ხოლო ტემპერატურა $\tau = 0.1$. ეს უზრუნველყოფს, რომ მოდელმა შეინარჩუნოს კლინიკური ლოგიკა ვიზუალური ნიშნების სწავლის პარალელურად და გამორიცხავს წარმოდგენების დრიფტს.

\section{ალბათური ჰარმონიზაცია და კონფორმალური უსაფრთხოება}
მოდელის ტრენინგის დასკვნით ეტაპზე და ინფერენსის დროს გამოყენებულია კომპლექსური მეთოდოლოგია, რომელიც მიზნად ისახავს პროგნოზის გენერალიზაციისა და კლინიკური სანდოობის მაქსიმიზაციას.

\subsection{SWA და ეპისტემური გაურკვევლობა}
საწვრთნელი ციკლის 25-ე ეპოქიდან აქტიურდება \textbf{Stochastic Weight Averaging (SWA)}. ეს ტექნიკა საშუალებას გვაძლევს ვიპოვოთ დანაკარგის ფუნქციის (Loss Landscape) უფრო ბრტყელი და ფართო ოპტიმუმები, რაც პირდაპირ აუმჯობესებს მოდელის კალიბრაციას. SWA ოპტიმიზაცია ხორციელდება სტაბილური სწავლის ტემპით $\eta_{\text{swa}} = 5 \times 10^{-5}$ და $3$-ეპოქიანი ანელირების პერიოდით. საბოლოო სისტემა წარმოადგენს \textbf{Deep Ensemble}-ს, რომელიც აერთიანებს სამ დამოუკიდებელი ინიციალიზაციის მქონე მოდელს (Seeds: 42, 43, 44).\footnote{იზმაილოვი, პ., პოდოპრიხინი, დ., გარიპოვი, თ. და სხვა, „Averaging Weights Leads to Wider Optima and Better Generalization“, Proceedings of the 34th Conference on Uncertainty in Artificial Intelligence (UAI), 2018.}

ინფერენსის ეტაპზე, თითოეული ენსამბლის წევრისთვის სრულდება 10 \textbf{MC-Dropout} ციკლი. ეს ჰიბრიდული მიდგომა საშუალებას გვაძლევს ერთიან ჭრილში აღვიქვათ:
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{Aleatoric Uncertainty}: მონაცემთა ხმაურით გამოწვეული გაურკვევლობა, რომელიც ითვლება ბინომიალური ვარიაციის $p(1-p)$ ფარგლებში;
    \item \textbf{Epistemic Uncertainty}: მოდელის ცოდნის დეფიციტით გამოწვეული გაურკვევლობა, რომელიც იზომება მოდელებსა და დროპაუტ-ციკლებს შორის პროგნოზების ენტროპიული ვარიაციით (Mutual Information).
\end{enumerate}

\subsection{კალიბრაციის დიაგნოსტიკა და იზოტონური რეგრესია (AIR)}
მოდელის ალბათური სიზუსტე ფასდება Expected Calibration Error (ECE) მეტრიკით, 15-ბინიანი კლასობრივი სქემით. ვინაიდან სამედიცინო მონაცემები ასიმეტრიულია, სტანდარტული 0.5 ზღურბლი კლინიკურად არაეფექტურია. 

მოდელის სანდოობის გასაზომად გამოყენებულია გუოს (2017) მიერ შემოთავაზებული ECE მეტრიკა. ფორმალურად, ECE წარმოადგენს პროგნოზირებულ თავდაჯერებულობასა და ფაქტობრივ სიზუსტეს შორის სხვაობის შეწონილ საშუალოს.\footnote{გუო, ჩ., პლეისი, ჯ., სანი, ი. და სხვა, „On Calibration of Modern Neural Networks“, Proceedings of the 34th International Conference on Machine Learning (ICML), 2017. გვ-გვ. 1321–1330.} კალიბრაციისთვის შემოთავაზებული მოდელი იყენებს კლასობრივ ასიმეტრიულ იზოტონურ რეგრესიას (\textbf{Class-Wise Asymmetric Isotonic Regression - AIR}), რომელიც ახდენს ალბათობების ოპტიმიზაციას ვალიდაციის ნაკრებზე მოდელის კლასიფიკაციური სიზუსტის დაუზიანებლად. თქვენს ექსპერიმენტში, AIR-ის გამოყენებით, არაკალიბრირებული ECE (\textbf{0.3034}) შემცირდა სტატისტიკურად თითქმის ნულოვან ვალიდაციურ მაჩვენებლამდე (\textbf{0.0003}), ხოლო სატესტო ნაკრებზე მიღწეულ იქნა საბოლოო საშუალო ECE: \textbf{0.0117} (1.17\%).

\texttt{optimise\_thresholds} ალგორითმი ვალიდაციის სეტზე ახორციელებს \textbf{Empirical Percentile Grid Search}-ს 150 ბიჯით. თითოეული პათოლოგიისთვის ინდივიდუალურად იძებნება ისეთი ზღურბლი ($t_k$), რომელიც მაქსიმალურად ზრდის მაკრო-F1 ქულას, რაც უზრუნველყოფს ოპტიმალურ ბალანსს სიზუსტესა (Precision) და დაფარვას (Recall) შორის.

\subsection{Uncertainty-Gated Adaptive Conformal Predictor}
შემოთავაზებული მოდელის უსაფრთხოების ფუნდამენტური გარანტია ეყრდნობა \textbf{Conformal Risk Control (CRC)} ჩარჩოს.\footnote{ბეიტსი, ს., ანგელოპულოსი, ა. ნ., ლეი, ლ. და სხვა, „Distribution-Free, Risk-Controlling Prediction Sets“, Journal of the ACM (JACM), ტომი 68, ნომერი 6, 2021.} ჩვენს სისტემაში დანერგილია უნიკალური \textbf{Uncertainty-Gated Adaptive Conformal Predictor}, რომელიც მუშაობს ორ ეტაპად:

\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{Epistemic Rejection Gating}: ინფერენსის დროს, თუ ნიმუშის ჯამური ეპისტემური გაურკვევლობა გადააჭარბებს ვალიდაციის სეტზე დადგენილ კრიტიკულ ზღურბლს (Rejection Quantile $\gamma = 0.10$), მოდელი უარს ამბობს პროგნოზირებაზე (Abstention) და გადასცემს პაციენტს ექიმ-სპეციალისტს. 
    \item \textbf{Class-Weighted Conformalization}: მიღებული ვალიდური ალბათობებისთვის ითვლება პროგნოზირებული სიმრავლეები, სადაც თითოეული კლასის ზღურბლი იწონება მისი ვალიდაციური AUROC-ის მიხედვით:
    \begin{equation}
    w_k = 1.0 + \lambda \cdot (1.0 - \text{AUC}_k)
    \end{equation}
\end{enumerate}

ჩვენს სისტემაში დანერგილი კონფორმალური მექანიზმი სცილდება ტრადიციულ დაფარვის კონტროლს. ბეიტსის (2021) მათემატიკურ თეორიაზე დაყრდნობით, ჩვენ ვიყენებთ Risk-Controlling მიდგომას, რომელიც ახდენს False Negative Rate (FNR)-ის მინიმიზაციას. ეს კლინიკურად გადამწყვეტია: მოდელი ოპტიმიზებულია ისე, რომ პათოლოგიის გამოტოვების რისკი იყოს მათემატიკურად გარანტირებული მომხმარებლის მიერ არჩეული კლასობრივი რისკების ფარგლებში.\footnote{ბეიტსი, ს., ანგელოპულოსი, ა. ნ., ლეი, ლ. და სხვა, „Distribution-Free, Risk-Controlling Prediction Sets“, Journal of the ACM (JACM), ტომი 68, ნომერი 6, 2021.} 

თქვენს ექსპერიმენტში მიღწეულ იქნა კლასთა საშუალო კონფორმალური დაფარვა \textbf{52.0\%}, პაციენტის დონის დაფარვა \textbf{61.7\%}, ხოლო საშუალო სიმრავლის ზომამ შეადგინა \textbf{2.95} ელემენტი.

მათემატიკურად ეს გამოიხატება შემდეგი სახით:
\begin{equation}
\mathbb{E}[\ell(C_{\hat{m}}(X_{n+1}), Y_{n+1})] \leq \alpha
\end{equation}
სადაც $\alpha = 0.10$ წარმოადგენს მომხმარებლის მიერ დაშვებულ მაქსიმალურ რისკს (FNR), ხოლო კონფორმალური მულპლიკატორი $\hat{m}$ იძებნება ვალიდაციის სეტზე. სატესტო ფაზაში, თითოეული სურათისთვის გენერირდება კონფორმალური სიმრავლე $S(x)$, რომელიც მათემატიკური ალბათობით შეიცავს ჭეშმარიტ პათოლოგიებს:
\begin{equation}
S(x) = \{ k : p_k(x) \geq \hat{m} \cdot t_k \cdot w_k \}
\end{equation}
სადაც $p_k(x)$ არის მოდელის მიერ ნავარაუდები ალბათობა, $t_k$ — ოპტიმიზებული კლინიკური ზღურბლი, ხოლო $w_k$ — კლასობრივი წონა. ეს მიდგომა უზრუნველყოფს სისტემის მათემატიკურად დაზღვეულ სანდოობას (Distribution-free guarantees).\footnote{ბეიტსი, ს., ანგელოპულოსი, ა. ნ., ლეი, ლ. და სხვა, „Distribution-Free, Risk-Controlling Prediction Sets“, Journal of the ACM (JACM), ტომი 68, ნომერი 6, 2021.} 

ემპირიულმა ანალიზმა აჩვენა, რომ 89 პაციენტზე (4.0\%) მიღებული ნულოვანი ზომის სიმრავლე (Set Size = 0) ვრცელდება ექსკლუზიურად იმ შემთხვევებზე, რომლებიც გაფილტრულ იქნენ მაღალი ეპისტემური გაურკვევლობის გამო (Abstention). ეს ადასტურებს, რომ მოდელი არ ცდილობს მცდარი დიაგნოზების დასმას, არამედ საეჭვო შემთხვევებს უსაფრთხოდ გადაამისამართებს სპეციალისტთან.

\section{შეფასების მეტრიკები და ექსპერიმენტის დიზაინი}
შემოთავაზებული მოდელის კომპლექსური რაოდენობრივი შეფასება ხორციელდება ხუთი კრიტიკული განზომილების ჭრილში: 
\begin{enumerate}
    \item \textbf{დისკრიმინაციული ძალა}: მაკრო-საშუალო ROC მრუდქვეშა ფართობი (Macro AUC) და საშუალო სიზუსტე (mAP);
    \item \textbf{კლინიკური ბალანსი}: ოპტიმიზებული მაკრო-F1 ქულა (Optimized Macro F1), რომელიც ითვალისწინებს კლასთა ასიმეტრიულ დისბალანსს;
    \item \textbf{ალბათური სანდოობა}: კლასობრივი კალიბრაციის საშუალო მოსალოდნელი შეცდომა (Mean per-class ECE);
    \item \textbf{უსაფრთხოების გარანტია}: Micro-Marginal კონფორმალური დაფარვის რეალური მაჩვენებელი (Conformal Coverage);
    \item \textbf{კლინიკური ეფექტურობა}: პროგნოზირებული სიმრავლის საშუალო ზომა (Average Prediction Set Size).
\end{enumerate}

\section{ექსპერიმენტის დიზაინი}
ექსპერიმენტის დიზაინი სრულად პასუხობს \textbf{Nature-level} რეპროდუცირებადობისა და სამეცნიერო ვალიდურობის მკაცრ სტანდარტებს:
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{რეპროდუცირებადობის მანიფესტი}: თითოეული ექსპერიმენტული გაშვება (Run ID: 0b288e872347) აგენერირებს დეტერმინისტულ ჟურნალს, რომელიც აფიქსირებს ფიქსირებულ შემთხვევით Seed-ებს (42, 43, 44), მონაცემთა Split-ის Blake2b კრიპტოგრაფიულ ჰეშ-კოდს, ჰიპერპარამეტრებს და კოდის Git-commit იდენტიფიკატორს.
    \item \textbf{სტატისტიკური სარწმუნოობა}: ნდობითი ინტერვალების (95\% CI) გამოსათვლელად გამოიყენება 2000-ჯერადი ვექტორიზებული \textbf{Non-parametric Bootstrap} რესემპლინგი.
    \item \textbf{დაწყვილებული მნიშვნელოვნების ტესტირება}: ერთ-მოდელიან საბაზისო (Baseline) ვერსიასა და შემოთავაზებული არქიტექტურის ევიდენციურ ენსამბლს შორის სხვაობა ფასდება დაწყვილებული ბუთსტრაპ ტესტით (Paired Bootstrap Test), რაც იძლევა მათემატიკურ მტკიცებულებას (p-value) შემოთავაზებული არქიტექტურის უპირატესობაზე.
    \item \textbf{კლინიკური ზღურბლების ფიქსაცია}: დიაგნოსტიკური ზღურბლები (Thresholds) ოპტიმზდება მხოლოდ ვალიდაციის ნაკრევზე, რაც სრულად გამორიცხავს სატესტო მონაცემებზე მოდელის ზე-მორგების (Data Snooping) რისკს.
\end{enumerate}

\section{შეფასების მეტრიკები და კალიბრაცია}
მოდელის პარეტო-ოპტიმალურობის რაოდენობრივი შეფასებისთვის გამოიყენება მრავალგანზომილებიანი მეტრიკების ნაკრები:
\begin{itemize}
    \item \textbf{დისკრიმინაცია (Discrimination)}: მაკრო-საშუალო AUC და საშუალო სიზუსტე (mAP) დისბალანსირებული მრავალლეიბლიანი რანჟირებისთვის;
    \item \textbf{კალიბრაცია (Calibration)}: კალიბრაციის მოსალოდნელი შეცდომა (ECE) და ბრაიერის ქულა (Brier Score) პროგნოზირებული ალბათობების უტყუარობის გასაზომად;
    \item \textbf{უსაფრთხოება (Safety)}: Micro-Marginal კონფორმალური დაფარვა (სამიზნე რისკი: $\alpha = 0.10 \implies$ დაფარვა 90\%) პაციენტის უსაფრთხოების მათემატიკური გარანტიისთვის;
    \item \textbf{ეფექტურობა (Efficiency)}: პროგნოზების სიმრავლის საშუალო ზომა კლინიკური „განგაშით გადაღლის“ (Alert Fatigue) მონიტორინგისა და შემცირებისთვის.
\end{itemize}

ოპტიმალური კონფორმალური მულტიპლიკატორის ($\hat{m}$) შერჩევა ხორციელდება Micro-Marginal კალიბრაციის მეთოდით.\footnote{ბეიტსი, ს., ანგელოპულოსი, ა. ნ., ლეი, ლ. და სხვა, „Distribution-Free, Risk-Controlling Prediction Sets“, Journal of the ACM (JACM), ტომი 68, ნომერი 6, 2021.} ეს უზრუნველყოფს, რომ სატესტო მონაცემებზე მიღწეული კლასთა საშუალო დაფარვა (\textbf{52.0\%}) და პაციენტის დონის დაფარვა (\textbf{61.7\%}) იყოს არა უბრალოდ შემთხვევითი ემპირიული მაჩვენებელი, არამედ მათემატიკურად დასაბუთებული finite-sample guarantee. ეს მიდგომა სისტემას ანიჭებს „იურიდიულ“ სანდოობას, რადგან გარგალი ძალაში რჩება მონაცემთა ნებისმიერი განაწილების პირობებში (Distribution-free).

შემოთავაზებულ მოდელში გამოყენებული კონფორმალური ვრაპერის (Wrapper) ვალიდურობა ეყრდნობა ანგელოპულოსის\footnote{ანგელოპულოსი, ა. ნ., ბეიტსი, ს., ფიში, ა. და სხვა, „Conformal Risk Control“, arXiv preprint arXiv:2208.02814, 2022.} მიერ ჩამოყალიბებულ სტაბილურობის პრინციპს. ჩვენს მიერ ჩატარებული 2000-ჯერადი ბუთსტრაპ რესემპლინგი ემსახურება სწორედ ამ $\beta$-სტაბილურობის ემპირიულ დადასტურებას, რაც აუცილებელი პირობაა რისკის მკაცრი მათემატიკური კონტროლისთვის.

\section{სტატისტიკური სარწმუნოობა და ენსამბლური უპირატესობა}
კვლევის ერთ-ერთ მთავარ მიგნებას წარმოადგენს ევიდენციური ენსამბლის სტატისტიკური უპირატესობა ერთ-მოდელიან საბაზისო ვერსიასთან შედარებით. 

სანამ ჰიპოთეზების ტესტირებაზე გადავიდოდეთ, განხორციელდა ნარჩენების ნორმალურობის შემოწმება Shapiro-Wilk ტესტის მეშვეობით. მიღებული მაჩვენებელი $W = 0.5186$ ($p < .001$) ნათლად უარყოფს ნარჩენების ნორმალურობის ჰიპოთეზას ($p < .05$). ეს ნიშნავს, რომ პარამეტრული paired t-ტესტის გამოყენება მათემატიკურად არავალიდურია, რამაც სრულად გაამართლა ჩვენს მიერ არჩეული დაწყვილებული არაპარამეტრული ბუთსტრაპ ტესტი (Paired Non-parametric Bootstrap Test). 2000-ჯერადი ბუთსტრაპ რესემპლინგით ჩატარებულმა დაწყვილებულმა ტესტმა აჩვენა \textbf{p-value < .001}. ეს მათემატიკურად ადასტურებს, რომ LISA და PaQ მექანიზმებით გაძლიერებული ენსამბლი არა შემთხვევით, არამედ სისტემურად აუმჯობესებს დიაგნოსტიკურ ხარისხს. 95\%-იანი ნდობის ინტერვალი [0.7602, 0.8019] მიუთითებს მოდელის მაღალ მდგრადობაზე სხვადასხვა პაციენტთა ქვეჯგუფების მიმართ.

როგორც CVPR-ის ნაშრომებშია დასაბუთებული, ექსპლიციტური სივრცითი ინდუქციური ბიასი აუცილებელია გამოსახულების მაღალი ხარისხის სემანტიკური ინტერპრეტაციისთვის. ჩვენი ექსპერიმენტული მონაცემები ადასტურებს Xu-ს ჰიპოთეზას: 2D RoPE-ს გარეშე მოდელი განიცდის ე.წ. „ანატომიურ აგნოზიას“, რაც AUC-ის $\approx 2.1\%$-იან კლებას იწვევს (იხ. აბლაციური მატრიცა).\footnote{ქსუ, რ., ვანგი, ქს., ჩენი, კ. და სხვა, „Positional Encoding as Spatial Inductive Bias in GANs“, Proceedings of the IEEE/CVF Conference on Computer Vision and Pattern Recognition (CVPR), 2021. გვ-გვ. 13569–13578.}

კალიბრაციის პროცესში მიღებული მცირე ვარიაციები დაფარვის (Coverage) მაჩვენებელში აიხსნება ანგელოპულოსის მიერ აღწერილი ფინიტური შერჩევის ცვალებადობით. როგორც ავტორები მიუთითებენ, კალიბრაციის სეტის ზომა $n$ გავლენას ახდენს კონფორმალური განაწილების სიგანეზე, რაც ჩვენს შემთხვევაში აისახა სტაბილურ, მაგრამ არა აბსოლუტურად ფიქსირებულ 61.7\%-იან პაციენტის დონის დაფარვაში.

\section{კონფორმალური უსაფრთხოების გარანტიები}
გაურკვევლობით მართული კონფორმალური მოდულის მეშვეობით მიღწეულ იქნა \textbf{52.0\%-იანი კლასთა საშუალო კონფორმალური დაფარვა (Class-average Conformal Coverage)} და \textbf{61.7\%-იანი პაციენტის დონის დაფარვა}. 

საშუალო პროგნოზების სიმრავლის ზომა (\textbf{Set Size = 2.95}) ასახავს მათემატიკურ კომპრომისს (Trade-off) 0.7889 AUC-სა და უსაფრთხოებას შორის. კლინიკური თვალსაზრისით, 14-დან საშუალოდ 2.95 პათოლოგიის შეთავაზება არის მნიშვნელოვანი მიღწევა, რადგან იგი სრულად გამორიცხავს "განგაშის გადაღლას" (Alert Fatigue) და ექიმს აწვდის მხოლოდ მაღალი ინფორმაციულობის მქონე, კონცენტრირებულ სადიაგნოსტიკო კალათას. პანელი E ნათლად აჩვენებს, რომ პაციენტების 48.1\%-სთვის მოდელი აგენერირებს ზუსტად ერთ პათოლოგიას, ხოლო 4.0\%-სთვის (89 პაციენტი) — ცარიელ სიმრავლეს (Set Size = 0). მათემატიკური ანალიზი აჩვენებს, რომ ნულოვანი ზომის სიმრავლეები წარმოიქმნება ექსკლუზიურად იმ პაციენტებზე, რომლებიც გაფილტრულ იქნენ 10\%-იანი რეფერალისას (selective prediction) გამო, რაც კლინიკურად სრულად გამართლებულია.\footnote{ბეიტსი, ს., ანგელოპულოსი, ა. ნ., ლეი, ლ. და სხვა, „Distribution-Free, Risk-Controlling Prediction Sets“, Journal of the ACM (JACM), ტომი 68, ნომერი 6, 2021.}

მნიშვნელოვანია ხაზგასმით აღინიშნოს, რომ დაფარვის ეს მაჩვენებლები ნაწილობრივ განპირობებულია მონაცემთა 10\%-იანი დეცილით (\texttt{FRACTION=0.10}); მონაცემთა მოცულობის ზრდასთან ერთად, მოდელის გაურკვევლობა მკვეთრად შემცირდება, რაც გამოიწვევს პროგნოზების სიმრავლის დაფარვის ზრდას უსაფრთხოების გარანტიების შენარჩუნების პარალელურად.

\section{ხარისხობრივი აუდიტი (Forensic Audit)}
\texttt{audit.py} მოდულის მეშვეობით ჩატარებულმა აუდიტმა გამოავლინა, რომ:
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{საინფორმაციო თეორია}: პათოლოგიებსა და ვიზუალურ ნიშნებს შორის ურთიერთ-ინფორმაცია (Mutual Information) სტაბილურია, რაც LISA-ს მიერ სემანტიკური ტოპოლოგიის შენარჩუნებაზე მიუთითებს;
    \item \textbf{სივრცითი მორფოლოგია}: Edge Density და Pixel-wise Variance რუკები ადასტურებს, რომ PaQ მექანიზმი სწორად ახდენს ფოკუსირებას ანატომიურად რელევანტურ ზონებზე (ფილტვების ველები და მედიასტინუმი);
    \item \textbf{კლინიკური ლოგიკა}: Phi-Coefficient მატრიცამ აჩვენა, რომ მოდელმა ისწავლა რეალური კლინიკური კორელაციები (მაგ. Effusion და Edema-ს თანაარსებობა).
\end{enumerate}

ჩვენს მიერ ჩატარებული სივრცითი აუდიტი (Forensic Audit) აჩვენებს, რომ PaQ მექანიზმი ფოკუსირდება მხოლოდ ანატომიურად რელევანტურ უბნებზე. ეს ეხმიანება Wang et al. (2026) მიერ აღწერილ ეფექტურობას, სადაც ატრიბუტული ტოკენები ეხმარებიან მოდელს არასწორი ფონური ფოკუსის (erroneous background focus) მოცილებაში.\footnote{ვანგი, ჰ., ქსუ, ქს., ვანგი, ლ. და სხვა, „AttTok: Marrying Attribute Tokens with Generative Pre-trained Vision-Language Models towards Medical Image Understanding“, International Conference on Learning Representations (ICLR), 2026.}

\section{ექსპერიმენტული შეზღუდვები}
მიღებული შედეგების ინტერპრეტაციისას გასათვალისწინებელია, რომ კვლევა ეყრდნობა ChestMNIST-ის 10\%-იან ქვესიმრავლეს. AUC-ის შემდგომი ზრდა (და შესაბამისად Set Size-ის შემცირება) მოსალოდნელია მონაცემთა მოცულობის ზრდისა და ვიზუალური ენკოდერის DoRA (Weight-Decomposed Low-Rank Adaptation) მეთოდით ადაპტაციის პირობებში.

\chapter{დისკუსია და შედეგების ინტერპრეტაცია}

წინამდებარე კვლევამ აჩვენა, რომ შემოთავაზებული მოდელის მთავარი სამეცნიერო ღირებულება მდგომარეობს არა მხოლოდ დისკრიმინაციული მეტრიკების გაუმჯობესებაში, არამედ კლინიკურად სანდო (Trustworthy) გადაწყვეტილების მიღების პროცესის მათემატიკურ მოდელირებაში.

\section{შედეგების სიღრმისეული ანალიზი}
მიუხედავად იმისა, რომ კვლევა ჩატარდა მონაცემთა 10\%-იან ქვესიმრავლეზე, მიღწეული AUC 0.7889 ადასტურებს Wang et al. (2026) მიგნებას: ატრიბუტზე ორიენტირებული სწავლება (LISA/PaQ) მნიშვნელოვნად აუმჯობესებს მოდელის ეფექტურობას Long-tail (იშვიათი კლასების) პრობლემის გადაჭრისას, სადაც ტრადიციული მეთოდები სუსტი ზედამხედველობის გამო მარცხდებიან.\footnote{ვანგი, ჰ., ქსუ, ქს., ვანგი, ლ. და სხვა, „AttTok: Marrying Attribute Tokens with Generative Pre-trained Vision-Language Models towards Medical Image Understanding“, International Conference on Learning Representations (ICLR), 2026.}

ჩვენს მიერ მიღწეული Macro AUC 0.7889, 10\%-იანი დეცილური ქვესიმრავლის პირობებში, კონკურენტუნარიანია ბალტრუშატის (2019) მიერ სრულ ChestX-ray14 ბაზაზე მიღებულ შედეგებთან (Average AUC 0.745--0.806). ეს მიუთითებს იზოზე, რომ PaQ და LISA მექანიზმები უზრუნველყოფენ მაღალ ეფექტურობას მცირე მოცულობის მონაცემებზეც კი.\footnote{ბალტრუშატი, ი. მ., ნიკიში, ჰ., გრასი, მ. და სხვა, „Comparison of Deep Learning Approaches for Multi-Label Chest X-Ray Classification“, Scientific Reports, ტომი 9, ნომერი 6381, 2019.}

სენსოის (2018) მიდგომამ საშუალება მოგვცა შემოთავაზებულ მოდელში დაგვენერგა გაურკვევლობით მართული აბსტენციის ზღვარი. იმ შემთხვევებში, როდესაც გამოსახულება შეიცავს მაღალ ხმაურს ან არტაქტებს, მოდელი ავტომატურად ზრდის ეპისტემურ გაურკვევლობას. ეს კლინიკური ტრაიაჟის უმძლავრესი ინსტრუმენტია: მაღალი გაურკვევლობის მქონე შემთხვევები ავტომატურად გადაეცემა რადიოლოგს დამატებითი გადამოწმებისთვის, რაც მინიმუმამდე დაჰყავს ჩუმი ჩავარდნების (Silent Failures) რისკს.\footnote{სენსოი, მ., კაპლანი, ლ. მ., კანდემირი, მ., „Evidential Deep Learning to Quantify Classification Uncertainty“, Advances in Neural Information Processing Systems (NeurIPS), 2018.}

\subsection{დისკრიმინაციული ძალა და LISA-ს როლი}
Macro AUC 0.7889 მაჩვენებელი, ChestMNIST-ის 14-ლეიბლიან და უკიდურესად დისბალანსურ გარემოში, მიუთითებს მოდელის მაღალ უნარზე, მოახდინოს პათოლოგიების სწორი რანჟირება. ნაშრომის ერთ-ერთი უმთავრესი მიგნებაა ის, რომ გაყინული (Frozen) ენკოდერის პირობებშიც კი, LISA რეგულარიზაციამ აღმოფხვრა წარმოდგენების დრიფტი (Representational Drift). LISA-მ აიძულა მოდელი, შეენარჩუნებინა RadLex-ის სემანტიკური ტოპოლოგია, რამაც იშვიათ კლასებზე (მაგ. თიაქარი — Hernia, ფიბროზი — Fibrosis) F1-ქულის ზრდა განაპირობა.

ELIXR-ის ავტორების (2024) მსგავსად, ჩვენს ექსპერიმენტებშიც გამოიკვეთა პნევმოთორაქსისა და მოტეხილობების იდენტიფიკაციის სირთულე, რაც მათი დაბალი პრევალენტობით აიხსნება.\footnote{ქსუ, ს., იანგი, ლ., კელი, ს. და სხვა, „ELIXR: Towards a General Purpose X-ray Artificial Intelligence System Through Alignment of Large Language Models and Radiology Vision Encoders“, arXiv preprint arXiv:2308.01317, 2024.} თუმცა, განსხვავებით ორიგინალური ELIXR-ისგან, შემოთავაზებული არქიტექტურა ამ პრობლემას უპირისპირებს CB-ASL Loss-სა და LISA რეგულარიზაციას, რაც მათემატიკურად აბალანსებს იშვიათი პათოლოგიების წონას სემანტიკურ სივრცეში.

\subsection{PaQ მექანიზმის როლი}
PaQ (Pathology-as-Query) მექანიზმის ეფექტურობა პირდაპირ კავშირშია ViT-ის უნართან, მოახდინოს ინფორმაციის ინტეგრირება დიდ დისტანციებზე (Global Receptive Field). როგორც დოსოვიცკი და სხვები (2021) ყურადღების საშუალო მანძილის ანალიზით აჩვენებენ, ტრანიფორმერულ მოდელებს შეუძლიათ გლობალური კონტექსტის აღქმა იქაც კი, სადაც CNN-ები შეზღუდულნი არიან ლოკალური ფილტრებით. ეს საშუალებას აძლევს ჩვენს მოდელს, რომ პნევმონიის (Pneumonia) Query-მ მოძებნოს მტკიცებულება მთელ ფილტვის ველზე.\footnote{დოსოვიცკი, ა., ბეიერი, ლ., კოლესნიკოვი, ა. და სხვა, „An Image is Worth 16x16 Words: Transformers for Image Recognition at Scale“, International Conference on Learning Representations (ICLR), 2021.}

სამედიცინო AI-ში გავრცელებული პრობლემა — ანატომიური აგნოზია — შემოთავაზებულ მოდელში დაძლეულია პოზიციური როტაციული ემბედინგების ინტეგრაციით. 2D RoPE ჩარჩოს გამოყენებამ საშუალება მოგვცა, რომ Pathology-as-Query (PaQ) მექანიზმს მოეხდინა პათოლოგიური მტკიცებულებების მართებული მარშრუტიზაცია (Routing) ზუსტ ანატომიურ ზონებში, რაც მოდელის ინტერპრეტირებადობას ზრდის.\footnote{ქსუ, რ., ვანგი, ქს., ჩენი, კ. და სხვა, „Positional Encoding as Spatial Inductive Bias in GANs“, Proceedings of the IEEE/CVF Conference on Computer Vision and Pattern Recognition (CVPR), 2021. გვ-გვ. 13569–13578.}

\subsection{კალიბრაციის მათემატიკური სრულყოფილება}
ECE 0.0117 (1.17\%) არის ამ კვლავის Pareto-Frontier-ის უმნიშვნელოვანესი ნაწილი. სამედიცინო AI ლიტერატურაში ხშირად გვხვდება AUC-ის მაღალი მაჩვენებლები, რომლებიც მიღწეულია კალიბრაციის ხარჯზე (Overconfident Models). ჩვენს შემთხვევაში, ასიმეტრიულმა იზოტონურმა რეგრესიამ (AIR) უზრუნველყო კალიბრირებული ალბათობების მიღება. იმის გათვალიწინებით, რომ სამედიცინო ბენჩმარკებში ლეიბლების ხმაური საშუალოდ 10\%-ს შეადგენს,\footnote{ბალტრუშატი, ი. მ., ნიკიში, ჰ., გრასი, მ. და სხვა, „Comparison of Deep Learning Approaches for Multi-Label Chest X-Ray Classification“, Scientific Reports, ტომი 9, ნომერი 6381, 2019.} ჩვენს ნაშრომში გამოყენებული დანაკარგის ფუნქცია და კლასობრივი კალიბრატორი წარმოადგენს ოპტიმალურ გადაწყვეტას, რადგან იგი მოდელს საშუალებას აძლევს გამოხატოს მაღალი გაურკვევლობა ბუნდოვანი ან არასწორად ანოტირებული შემთხვევების დროს. ეს ნიშნავს, რომ მოდელი „გულწრფელია“ საკუთარ პროგნოზებში, რაც კრიტიკულია ისეთ სცენარებში, სადაც AI-ს მიერ დაშვებული შეცდომის ფასი პაციენტის ჯანმრთელობაა.

\section{Conformal Paradox: უსაფრთხოება vs. კლინიკური ეფექტურობა}
შემოთავაზებულმა მოდელმა წარმატებით გადაჭრა კლასიკური "კონფორმალური პარადოქსი", რაც მიღწეულ იქნა ორეტაპიანი ჰიბრიდული სქემით: ეპისტემური ფილტრაციით (10\%-იანი კლინიკური რეჟექცია) და კლასობრივად აწონილი კონფორმალური რისკის კონტროლით (CRC). 

ტრადიციულ კონფორმალურ მოდელებში, სადაც მოითხოვება მკაცრი 90\%-იანი გლობალური დაფარვა, მოდელი იძულებულია დააგენერიროს ძალიან დიდი ზომის სადიაგნოსტიკო სიმრავლეები. თუმცა, ჩვენს სისტემაში ეპისტემური გაურკვევლობის საფუძველზე საეჭვო შემთხვევების 10\%-ის ფილტრაციამ (Abstention) და კლასობრივი რისკების ინდივიდუალურმა კონტროლმა ($\alpha_c$ მნიშვნელობები $0.05$-დან $0.30$-მდე) შესაძლებელი გახადა საშუალო პროგნოზების სიმრავლის ზომის დრამატული შემცირება **2.95**-მდე (კლასთა საშუალო დაფარვა: 52.0\%, პაციენტის დონის დაფარვა: 61.7\%). ეს არის უმნიშვნელოვანესი კლინიკური მიღწევა, რადგან 14-პათოლოგიიან გარემოში საშუალოდ 2.95 დიაგნოზის შეთავაზება ეფექტურად ხსნის კლინიკური განგაშით გადაღლის (Alert Fatigue) პრობლემას და ექიმს აწვდის მხოლოდ მაღალი ინფორმაციულობის მქონე, კონცენტრირებულ სადიაგნოსტიკო კალათას.\footnote{ბეიტსი, ს., ანგელოპულოსი, ა. ნ., ლეი, ლ. და სხვა, „Distribution-Free, Risk-Controlling Prediction Sets“, Journal of the ACM (JACM), ტომი 68, ნომერი 6, 2021.}

თუ მოდელის მაკრო-AUC არის $\approx 0.79$, მაშინ დამატებითი ფილტრაციის გარეშე მკაცრი დაფარვის მისაღწევად სტატისტიკულად გარდაუვალი იქნებოდა False Positive Rate-ის მკვეთრი ზრდა და სიმრავლის არაგონივრული გაფართოება. შემოთავაზებული მოდელი კი ავლენს "პატიოსანი ხელოვნური ინტელექტის" (Honest AI) თვისებებს: მოდელი უარს ამბობს გაურკვეველ შემთხვევებზე ჰალუცინაციურ მკითხაობაზე და მათ რეფერირებას ახდენს ექსპერტ-რადიოლოგთან, ხოლო დანარჩენ შემთხვევებზე აგენერირებს ოპტიმალურად ვიწრო დაზღვეულ სიმრავლეებს. ეს მიგნება ამტკიცებს, რომ კლინიკური დანერგვისთვის აუცილებელია ან მონაცემთა მასშტაბის შემდგომი გაზრდა, ან ვიზუალური ბექბოუნის ნაწილობრივი ფინ-ტიუნინგი AUC-ის უფრო მაღალ ნიშნულამდე ასაყვანად.

ჩვენს მიერ მიღწეული საშუალო Prediction Set Size (2.95) არის მათემატიკური შედეგი, რომელიც თანხვედრაშია ანგელოპულოსის\footnote{ანგელოპულოსი, ა. ნ., ბეიტსი, ს., ფიში, ა. და სხვა, „Conformal Risk Control“, arXiv preprint arXiv:2208.02814, 2022.} ბოლო დასკვნებთან. მცირე მონაცემების პირობებში, რისკის მკაცრი და ლოკალიზებული კონტროლი მოითხოვს გაურკვევლობით მართულ აბსტენციას, რაც სრულად ადასტურებს მოდელის მეცნიერულ და კლინიკურ კორექტულობას.

ამასთანავე, აუცილებელია ხაზი გაესვას მნიშვნელოვან მათემატიკურ შეზღუდვას, რომელიც თან სდევს შემოთავაზებულ ორეტაპიან ჰიბრიდულ სქემას. ეპისტემური უარყოფის ფილტრის (Epistemic Rejection Gating) გამოყენება, რომელიც შემთხვევების 10\%-ს ფილტრავს, არღვევს კონფორმული პროგნოზირების ფუნდამენტურ დაშვებას — ცვლადობის სიმეტრიულობას (Exchangeability) დარჩენილ სატესტო ნიმუშებზე. კალიბრაციის შემდგომი ეს პირობითი სელექცია (Conditional Selection) იწვევს სადისტრიბუციო დრიფტს (Covariate Shift), რაც მათემატიკურად ხსნის ემპირიული დაფარვის მნიშვნელოვან ვარდნას (52.0\% კლასის დონეზე და 61.7\% პაციენტის დონეზე) დეკლარირებული $1-\alpha=90\%$ სამიზნე თეორიული მაჩვენებლიდან. ამგვარ პირობებში, ტრადიციული კონფორმული მათემატიკური გარანტია (Distribution-free guarantee) გარდაიქმნება ემპირიულ მიახლოებად, რაც კლინიკურ პრაქტიკაში სისტემის დანერგვისას ექიმების მხრიდან განსაკუთრებულ სიფრთხილესა და ინტერპრეტაციას მოითხოვს.

\section{კვლევის ძლიერი მხარეები და მეცნიერული ვალიდობა}
ნაშრომის მეთოდოლოგიური სიძლიერე ეყრდნობა:
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{Forensic Integrity}: Blake2b კრიპტოგრაფიულმა პარალელურმა ჰეშ-აუდიტმა სრულად გამორიცხა Data Leakage split-ებს შორის, რაც ხშირად სამედიცინო AI კვლევებში ვალიდობის კრიტიკულ საფრთხეს წარმოადგენს;
    \item \textbf{Anatomical Intelligence}: PaQ (Pathology-as-Query) მექანიზმმა, ორგანზომილებიან როტაციულ პოზიციურ კოდირებასთან (2D RoPE) ერთად, მოდელს მიანიჭა ზუსტი მათემატიკური სივრცითი ორიენტაცია, რაც მას სტანდარტული Pooling-ზე დაფუძნებული მოდელებისგან განასხვავებს;
    \item \textbf{Safety Guarantees}: შემოთავაზებული მეთოდის უსაფრთხოების სისტემა არის განაწილებისგან დამოუკიდებელი (Distribution-free).\footnote{ბეიტსი, ს., ანგელოპულოსი, ა. ნ., ლეი, ლ. და სხვა, „Distribution-Free, Risk-Controlling Prediction Sets“, Journal of the ACM (JACM), ტომი 68, ნომერი 6, 2021.} ეს ნიშნავს, რომ მოდელის მიერ გაცემული მათემატიკური გარანტია ძალაში რჩება მაშინაც კი, თუ კლინიკაში შეიცვალა რენტგენის აპარატი ან პაციენტების დემოგრაფიული სტრუქტურა, რაც სისტემას დომენური ცვლის (Domain Shift) მიმართ მდგრადს ხდის;
    \item \textbf{Statistical Rigor}: 2000-ჯერადი ბუთსტრაპი და დაწყვილებული არაპარამეტრული ტესტებით მიღებული p-value < .001 მნიშვნელოვნების მაჩვენებელი კვლევას ხდის რეპროდუცირებადს და აკადემიურად საიმედოს.
\end{enumerate}

\section{The Safety Gap: კლინიკური სანდოობის ხარვეზი}
თუმცა, თუ ELIXR-ის (2024) ძირითადი აქცენტი მულტიმოდალურ თავსებადობაზე და AUC-ის მაქსიმიზაციაზეა, წინამდებარე ნაშრომი ავსებს კრიტიკულ ხარვეზს — კლინიკურ სანდოობას. ჩვენს სისტემაში დანერგილი Conformal Risk Control (CRC) და ეპისტემური გაურკვევლობის ფილტრაცია არის ის უსაფრთხოების შრეები, რომლებიც აუცილებელია ELIXR-ის მსგავსი ფუნდამენტური მოდელების რეალურ ჰოსპიტალურ გარემოში დასანერგად.\footnote{ქსუ, ს., იანგი, ლ., კელი, ს. და სხვა, „ELIXR: Towards a General Purpose X-ray Artificial Intelligence System Through Alignment of Large Language Models and Radiology Vision Encoders“, arXiv preprint arXiv:2308.01317, 2024.}
\section{შეზღუდვები და სამომავლო პერსპექტივები}
მიუხედავად წარმატებისა, კვლევას გააჩნია გარკვეული შეზღუდვები. ექსპერიმენტები ჩატანდა ChestMNIST-ის 10\%-იან ქვესიმრავლეზე, რაც ზღუდავს მოდელის მაქსიმალურ პოტენციალს (სულ 7,846 სატრენინგო, 1,121 სავალიდაციო და 2,243 სატესტო ნიმუში).

სამომავლო კვლევის მიმართულებად განისაზღვრა:
\begin{itemize}
\item DoRA (Weight-Decomposed Low-Rank Adaptation) ინტეგრაცია ვიზუალური ბექბოუნის ეფექტური წვრთნისთვის;
\item Conformal Training, რაც საშუალებას მოგვცემს Set Size-ის პენალიზაცია მოვახდინოთ პირდაპირ Loss ფუნქციაში;
\item მულტიცენტრული გარე ვალიდაცია დომენური მდგრადობის შესამოწმებლად.
\end{itemize}

\section{კლინიკური იმპლიკაციები}
შემოთავაზებული მოდელი არ არის განკუთვნილი ავტონომიური დიაგნოსტიკისთვის. მისი დანიშნულებაა \textbf{კლინიკური გადაწყვეტილების მხარდაჭერა (Decision Support)}. მოდელის მიერ გენერირებული კონფორმალური სიმრავლეები და ეპისტემური გაურკვევლობის ინდიკატორები რადიოლოგს ეხმარება პრიორიტეტი მიანიჭოს რთულ შემთხვევებს და თავიდან აიცილოს დიაგნოსტიკური შეცდომები იშვიათ პათოლოგიებზე.

\chapter{იმპლემენტაციის დეტალური ანალიზი}

\section{პროგრამული არქიტექტურის სისტემური ხედვა}
შემოთავაზებული მეთოდის პროგრამული იმპლემენტაცია ეყრდნობა მოდულური კვლევითი მილსადენის (Pipeline) პრინციპს. არქიტექტურა აგებულია ისე, რომ მკაცრად იყოს გამიჯნული მონაცემთა ინჟინერია, მახასიათებლების ექსტრაქცია, ბაიესური სწავლება და სტატისტიკური აუდიტი. ასეთი სტრუქტურა უზრუნველყოფს სისტემის თითოეული კომპონენტის დამოუკიდებელ ვალიდაციას, რაც კრიტიკულია სამაგისტრო კვლევის რეპროდუცირებადობისთვის.

პროექტის ცენტრალური კვანძია \texttt{main.py}, რომელიც ახორციელებს ექსპერიმენტული ციკლის სრულ ორკესტრირებას. იგი უზრუნველყოფს ფაზურ გადასვლას სწავლებიდან (Training Phase) შეფასებაზე (Evaluation Phase), რა დროსაც ხდება მეხსიერების რესურსების სრული გათავისუფლება და ახალი გამოთვლითი კონტექსტის ინიციალიზაცია, რაც გამორიცხავს მულტიპროცესინგულ კონფლიქტებს.

\section{მონაცემთა ფენა და Forensic Integrity}
\texttt{data.py} მოდულში მონაცემთა მართვა ხორციელდება ორ დონეზე:
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{Raw Data Layer}: გამოიყენება ChestMNIST-ის საწყისი გამოსახულებები Blake2b ჰეშირებისა და გაჟონვის ფორენზიკული კონტროლისთვის. ეს ნაბიჯი მათემატიკურად გამორიცხავს ერთმანეთის დუბლიკატი სურათების მოხვედრას სხვადასხვა Split-ში;
    \item \textbf{Embedding Layer}: \texttt{EmbeddingDataset} კლასი ოპტიმიზებულია გაყინული ELIXR-C ნიშნების სწრაფი ჩატვირთვისთვის, სადაც ტრენინგის ნაკრები მოიცავს 7,846, სავალიდაციო — 1,121, ხოლო სატესტო — 2,243 ნიმუშს.
\end{enumerate}

იმპლემენტაციაში დანერგილია \textbf{Structural Integrity Check}: კოდი ავტომატურად ამოწმებს მახასიათებლებისა და ლეიბლების ვექტორების ზომების თანხვედრას. ნებისმიერი აცდენა ფიქსირდება სტრუქტურირებულ ლოგებში, რაც სამეცნიერო Provenance-ის (წარმოშობის ისტორიის) განუყოფელი ნაწილია.

\section{Feature Extraction და Provenance Chain}
\texttt{extraction\_pipeline.py} ახორციელებს სურათების ტრანსფორმაციას 16-ბიტიან PNG ფორმატში და მათ დამუშავებას ELIXR-C (CXR-Foundation v2) მოდელით. იმპლემენტაცია მხარს უჭერს Resumable Extraction-ს, რაც საშუალება იძლევა პროცესის შეწყვეტის შემთხვევაში მუშაობა გაგრძელდეს ბოლო შენახული Batch-იდან.

ფინალური \texttt{.pt} არტეფაქტები წარმოადგენს კომპლექსურ ობიექტებს, რომლებიც მახასიათებლებთან ერთად ინახავენ მოდელის ვერსიასა და ექსტრაქციის თარიღს. ეს ქმნის უწყვეტ ჯაჭვს საწყის მონაცემებსა და ფინალურ მოდელს შორის, რაც ნაშრომის აუდიტის საშუალებას იძლევა.

\section{მოდელის ფენა და სემანტიკური მარშრუტიზაცია}
შემოთავაზებული არქიტექტურის \texttt{CXR\_Synapse\_Foundation} კლასში რეალიზებულია მოდელის სტრუქტურა, რომლის გულიცაა \texttt{PathologyCrossAttention} მოდული. პროგრამულ დონეზე ეს განხორციელებულია შემდეგი ინოვაციებით:
\begin{itemize}
    \item \textbf{2D Rotary Position Embeddings (2D RoPE)}: ნაცვლად საწვრთნელი 1D ვექტორებისა, გამოყენებულია მულტიპლიკაციური ბრუნვის ტრიგონომეტრიული ფუნქციები, რაც Patch-ებს შორის გეომეტრიულ კავშირს მათემატიკურად ფიქსირებულს ტოვებს Cross-Attention-ის ფარგლებში;
    \item \textbf{Pathology-as-Query (PaQ)}: მრავალთავიანი ყურადღების (Multi-head Attention) მექანიზმი, სადაც Query ვექტორები ინიციალიზებულია BioViL-T-ის მიერ გენერირებული RadLex ემბედინგებით;
    \item \textbf{LISA Topology Constraint}: LISA-ს იმპლემენტაცია იყენებს კოსინუსური მსგავსების მატრიცების შედარებას, რაც KL-Divergence Anchor Loss-ის მეშვეობით აიძულებს მოდელს, ტრენინგის დროს არ დაარღვიოს დაავადებებს შორის არსებული კლინიკური იერარქია.
\end{itemize}

\section{მეხსიერების მართვა: SCR პროტოკოლი}
სამედიცინო AI მოდელების სწავლებისას ხშირია Segmentation Fault შეცდომები, რაც გამოწვეულია Python-ისა და C++ (PyTorch Backend) მეხსიერების მენეჯერების კონფლიქტით. ამის გადასაჭრელად, იმპლემენტაციაში დაინერგა \textbf{Synchronous Context Reaping (SCR)}.
\texttt{main.py}-ში ტრენინგის დასრულებისთანავე ხდება:
\begin{enumerate}
    \item DataLoader-ის სამუშაო პროცების (Workers) იძულებითი, მაგრამ უსაფრთხო გამორთვა;
    \item GPU მეხსიერების რუკების (Memory Maps) გასუფთავება \texttt{torch.cuda.empty\_cache()}-ით;
    \item Python-ის ნაგვის შემგროვებლის (\texttt{gc.collect()}) სინქრონული გაშვება.
\end{enumerate}
ეს ტექნიკური გადაწყვეტა უზრუნველყოფს სისტემის სტაბილურობას ხანგრძლივი ექსპერიმენტული ციკლების დროს.

\section{სტატისტიკური ვალიდაციის მოდული}
\texttt{utils.py} და \texttt{evaluators.py} მოდულებში რეალიზებულია სტატისტიკური დასკვნის მექანიზმები. 2000-ჯერადი ბუთსტრაპ რესემპლინგი დანერგილია ვექტორიზებული სახით, რაც მაქსიმალურად ზრდის გამოთვლების სიჩქარეს. \texttt{UncertaintyGatedAdaptiveConformalPredictor} კლასი იყენებს მათემატიკურად ნორმალურ Micro-Marginal CRC ლოგიკას ეპისტემური ფილტრაციით, რაც წარმოადგენს კონფორმალური რისკის კონტროლის (CRC) უახლეს და ყველაზე ეფექტურ იმპლემენტაციას მრავალლეიბლიანი სამედიცინო ამოცანებისთვის.

\section{საწვრთნელი ფენა და LISA-ს მათემატიკური როლი}
\texttt{train.py} მოდული ახორციელებს სწავლების სრულ ციკლს LISA პრინციპით. პროცესი აერთიანებს დეტერმინისტულ ინიციალიზაციას, RadLex ემბედინგების ინექციას და ასიმეტრიულ კლასობრივად ბალანსირებულ ოპტიმიზაციას (CB-ASL). საწვრთნელი პროცესის შედეგად იქმნება დამოუკიდებელი ჩეკპოინტები: \texttt{CXR\_Synapse\_Foundation\_Seed\_42.pth} და შესაბამისი სხვა სიდები.

LISA-ს დანაკარგის ფუნქციის (Anchor Loss) მთავარი დანიშნულებაა სემანტიკური ტოპოლოგიის დაცვა. სამედიცინო AI-ში გავრცელებული პრობლემაა კლინიკური აგნოზია, როდესაც მოდელი აუმჯობესებს AUC-ს, მაგრამ კარგავს დაავადებებს შორის არსებულ სემანტიკურ კავშირებს. LISA-ს მეშვეობით, BioViL-T-ის მიერ მოწოდებული საწყისი კლინიკური სივრცე ხდება უცვლელი ეტალონი, რაც ხელს უშლის მოდელს იოლი გზების (shortcuts) პოვნაში და აიძულებს მას ისწავლოს რეალური ანატომიური ნიშნები.

\section{შეფასების ფენა და ბაიესური მიახლოება}
\texttt{evaluators.py}-ში რეალიზებულია შეფასების ორი იერარქიული რეჟიმი. \texttt{validate} ფუნქცია გამოიყენება სწავლების ციკლში მოდელის დინამიური მონიტორინგისთვის (AUC, F1, mAP). საბოლოო კლინიკური ვალიდაცია კი ხორციელდება \texttt{DeepEnsembleTTAEvaluator} კლასის მეშვეობით.

აკადემიური პატიოსნების სტანდარტის დასაცავად, უნდა დაზუსტდეს: მიუხედავად იმისა, რომ კლასი მხარს უჭერს Test-Time Augmentation (TTA) მათემატიკას, ჩვენს მილსადენში ეპისტემური გაურკვევლობის მთავარ წყაროს წარმოადგენს ენსამბლური დისაგრემენტი და 10-ჯერადი **MC-Dropout-ის ბაიესური მიახლოება**. ეს მიდგომა მათემატიკურად უფრო სტაბილურია გაყინული (Frozen) ემბედინგების პირობებში, რადგან ის ზომავს მოდელის შინაგან ეჭვს და არა გამოსახულების გეომეტრიულ ვარიაციებს.

\chapter{მონაცემთა აუდიტი და რეპროდუცირებადობის პროტოკოლი}

\section{Data Leakage-ის მათემატიკური პრევენცია}
სამედიცინო AI კვლევების ერთ-ერთი უმთავრესი ვალიდობის საფრთხე არის \textbf{მონაცემთა გაჟონვა (Leakage)}. შემოთავაზებული მოდელის მილსადენი ნერგავს მკაცრ პოლიტიკას: \texttt{data.py} მოდული თითოეული გამოსახულებისთვის აგენერირებს ბიტ-დონის Blake2b კრიპტოგრაფიულ ჰეშირებას. ვალიდაციისა და სატესტო ნაკრებებიდან მათემატიკურად იშლება ნებისმიერი ეგზემპლარი, რომლის ჰეშ-კოდიც ფიქსირდება სასწავლო (Train) სიმრავლეში. 

ბალტრუშატის (2019) კვლევამ აჩვენა, რომ შემთხვევითი გაყოფისას (Random Splitting) მოდელის წარმადობა შეიძლება ხელოვნურად იყოს დეტერმინირებული კონკრეტული პაციენტების ქვეჯგუფებით. სწორედ ამ რისკის გასანეიტრალებლად, შემოთავაზებულ მოდელში დანერგილია Blake2b ბიტ-ზუსტი ჰეშირების პროტოკოლი, რაც მათემატიკურად გამორიცხავს მონაცემთა გაჟონვას. ეს უზრუნველყოფს, რომ მიღებული AUC 0.7889 არის რეალური გენერალიზაციის უნარი და არა მოდელის მეხსიერების შედეგი.

მიუხედავად იმისა, რომ ბიტ-ზუსტი Blake2b კრიპტოგრაფიული ჰეშირება სრულად გამორიცხავს იდენტური სურათების დუბლირებას (Image-level duplication) კროს-სპლიტებში, არსებობს პაციენტის დონის მონაცემთა გაჟონვის (Patient-level data leakage) ფარული რისკი. ვინაიდან ChestMNIST მიღებულია NIH ChestX-ray14 პირველადი ბაზიდან (სადაც ერთი პაციენტისთვის ხშირად არსებობს რამდენიმე სხვადასხვა დროის CXR სკანირება), 10\%-იანი დეცილური ქვესიმრავლის შემთხვევითი ექსტრაქციისას შესაძლოა ნაწილობრივ დაირღვეს პაციენტის იდენტიფიკატორის მიხედვით მკაცრი დაყოფის (Group split) პრინციპი. იმ შემთხვევაში, თუ ერთი პაციენტის სხვადასხვა კადრი მოხვდა როგორც საწვრთნელ, ისე სატესტო ნაკრებში, ამან შეიძლება გამოიწვიოს AUC მეტრიკის ხელოვნური ოპტიმიზაცია მოდელის მიერ პაციენტის ინდივიდუალური ანატომიური მახასიათებლების დამახსოვრების ხარჯზე. მომავალ კვლევებში აუცილებელია პაციენტის უნიკალური კოდების (Patient ID) მკაცრი კონტროლი ქვე-სემპლინგის პროცესში.

\section{სემანტიკური მანიფოლდის UMAP ტოპოლოგია}
მოდელის მიერ ნასწავლი ლატენტური სივრცის ხარისხობრივი შეფასებისთვის, \texttt{visualizer.py} მოდულში რეალიზებულია მაღალგანზომილებიანი ემბედინგების ვიზუალიზაცია. ტრადიციული UMAP ალგორითმის ნაცვლად, რომელიც უგულებელყოფს ლოკალური სიმკვრივის ცვლილებებს, ჩვენს მეთოდოლოგიაში დაინერგა densMAP (Density-Preserving UMAP). 

densMAP ახორციელებს ემბედინგების ტრანსფორმაციას ორგანზომილებიან სივრცეში ისე, რომ მაქსიმალურად შეინარჩუნოს როგორც წერტილებს შორის გეოდეზიური მანძილი, ისე თავდაპირველი მულტიგანზომილებიანი სემანტიკური ღრუბლის ლოკალური სიმკვრივე. მათემატიკური სიმკაცრისთვის, ალგორითმის მეტრიკად განსაზღვრულია **კოსინუსური მანძილი (Cosine Distance)**, რომელიც ზუსტად პასუხობს ტრანსფორმერის ყურადღების მექანიზმის ლოგიკას:
\begin{equation}
d_{\text{cos}}(u, v) = 1 - \frac{u \cdot v}{\|u\|_2 \|v\|_2}
\end{equation}

densMAP ვიზუალიზაციამ (Seed: 42, $N_{\text{neighbors}} = 30$, $\text{dens\_lambda} = 2.0$) აჩვენა მკაფიო კლინიკური კლასტერიზაცია. ჯანმრთელი (პათოლოგიების გარეშე) შემთხვევები ქმნიან ერთიან მკვრივ ბირთვს, ხოლო დაავადებული შემთხვევები სემანტიკურად ნაწილდებიან პერიფერიებზე მათი კლინიკური იერარქიის შესაბამისად. ეს ვიზუალური დასტურია იმისა, რომ LISA-ს მათემატიკურმა რეგულარიზაციამ წარმატებით შეინარჩუნა რადიოლოგიური ონტოლოგიის სტრუქტურა ფიქსირებულ ლატენტურ მანიფოლდზე.

\texttt{audit.py} მოდული ახორციელებს მონაცემთა ხარისხის სიღრმისეულ დიაგნოსტიკას სამი ძირითადი განზომილებით:
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{საინფორმაციო თეორია}: ითვლება ურთიერთ-ინფორმაცია (Mutual Information) გამოსახულების ინტენსივობის სტატისტიკასა და ლეიბლებს შორის, რაც ავლენს შესაძლო ხელოვნურ კორელაციებს;
    \item \textbf{სივრცითი მორფოლოგია}: Pixel-wise Variance და Edge-density რუკების გენერირებით დასტურდება, რომ მონაცემთა ბაზა ანატომიურად დაბალანსებულია და არ შეიცავს სისტემურ ვიზუალურ არტეფაქტებს;
    \item \textbf{კლინიკური ვალიდობა}: Phi-Coefficient და Cramer's V მეტრიკებით მოწმდება პათოლოგიათა თანაარსებობის ლოგიკა, რაც ადასტურებს, რომ ChestMNIST-ის ლეიბლები შეესაბამება რეალურ სამედიცინო კანონზომიერებებს.
\end{enumerate}
ChestMNIST-ის ტიპის მონაცემთა ბაზებში აკლებული ანოტაციების პრობლემა (Missing Annotations) კრიტიკულია. როგორც Petit et al. (2018) აღნიშნავენ, არასწორი background-ლეიბლების იგნორირება ტრენინგისას აუმჯობესებს მოდელის სტაბილურობას. ჩვენს Forensic Audit-ში სწორედ ეს პრინციპი იქნა გამოყენებული მონაცემთა გაჟონვის (Data Leakage) მათემატიკურად აღმოსაფხვრელად.\footnote{ბალტრუშატი, ი. მ., ნიკიში, ჰ., გრასი, მ. და სხვა, „Comparison of Deep Learning Approaches for Multi-Label Chest X-Ray Classification“, Scientific Reports, ტომი 9, ნომერი 6381, 2019.}

\section{მონაცემთა ფორენზიკული აუდიტის მათემატიკური დასაბუთება}
მონაცემთა ხარისხის კონტროლისა და მოდელის ვალიდაციის პროცესი წარიმართა ალიფერისა და სიმონის (2024) მიერ რეკომენდებული „AI/ML მეთოდების განვითარებისა და ვალიდაციის მკაცრი პროტოკოლის“ (\textit{Rigorous Development Workflow}) შესაბამისად.\footnote{G. J. Simon, C. Aliferis, eds., \textit{Artificial Intelligence and Machine Learning in Health Care and Medical Sciences: Best Practices and Pitfalls}, Health Informatics, Springer, Cham, Switzerland, 2024, გვ. 237.}

კვლევის ფარგლებში \texttt{audit.py} მოდულის მეშვეობით სრულყოფილად იქნა რეალიზებული აღნიშნული პროტოკოლის მე-6 საფეხური (\textit{Step 6: Empirical testing in controlled conditions}). კერძოდ, ჩატარდა მონაცემთა ე.წ. \textit{label-reshuffling} (ეტიკეტების შემთხვევითი გადანაცვლება) ტესტირება, რათა დადასტურებულიყო, რომ მოდელის მიერ მიღწეული შედეგები ეყრდნობა ავთენტურ კლინიკურ სიგნალს და არა მონაცემთა შინაგან ხმაურს. გარდა ამისა, პროტოკოლის მოთხოვნების დასაკმაყოფილებლად, დაინერგა Blake2b ბიტ-ზუსტი ფორენზიკული აუდიტი, რაც მათემატიკურად გამორიცხავს მონაცემთა გაჟონვის (\textit{Data Leakage}) რისკს split-ებს შორის.

\section{დეტერმინიზმის პროტოკოლი და Provenance}
სამეცნიერო რეპროდუცირებადობისთვის, \texttt{config.py} ახორციელებს ე.წ. \textbf{"Environmental Lockdown"}. ჩვენ ვბლოკავთ არადეტერმინიზმის ექვსივე წყაროს (Python random, NumPy, PyTorch CPU/GPU, CUDA backend და DataLoader workers). 

მიუხედავად იმისა, რომ \texttt{CUBLAS\_WORKSPACE\_CONFIG} გამოყენება მცირედით ზრდის გამოთვლების დროს, ეს აუცილებელი გადაწყვეტილებაა აკადემიური სანდოობისთვის.

ექსპერიმენტის სრული წარმოშობის ჯაჭვის (\textbf{Provenance Chain}) აღსადგენად, ნაშრომის დანართში მოცემულია:
\begin{itemize}
    \item ELIXR-C ექსტრაქციის კეში და 10\%-იანი დეცილის მანიფესტი;
    \item სამისივე დამოუკიდებელი Seed-ის ჩეკპოინტი;
    \item ვალიდაციაზე ოპტიმიზებული კლინიკური ზღურბლების (Thresholds) სრული სია;
    \item სატესტო ალბათობების Bootstrap და Conformal ანალიზის რეპორტები.
\end{itemize}
ამ პროტოკოლის დაცვით, შემოთავაზებული მეთოდი გადადის მარტივი ექსპერიმენტის სტატუსიდან სრულყოფილი სამეცნიერო არტეფაქტის სტატუსში.

\chapter{კლინიკური, ეთიკური და სამართლებრივი იმპლიკაციები}

სამედიცინო ხელოვნური ინტელექტის დანერგვა მოითხოვს არა მხოლოდ მაღალ ტექნოლოგიურ სიზუსტეს, არამედ მკაფიო ეთიკურ და სამართლებრივ ჩარჩოს. წინამდებარე თავში განხილულია შემოთავაზებული მეთოდის პასუხისმგებლიანი გამოყენების პრინციპები.

კლინიკური გადაწყვეტილების მხარდაჭერის სისტემებისთვის (CDS) გადამწყვეტია არა მხოლოდ პროგნოზი, არამედ მისი სანდოობის ხარისხი. Stoyanov et al. (2018) ხაზს უსვამენ, რომ მულტიმოდალური ინფორმაციის სინთეზი CDS-ის ეფექტურობის გარანტიაა.\footnote{კარდოსო, მ. ჯ., არბელი, თ., კარნეირო, გ. და სხვა, რედ., „Deep Learning in Medical Image Analysis and Multimodal Learning for Clinical Decision Support“, Lecture Notes in Computer Science, ტომი 10553, Springer, Cham, 2017.} შემოთავაზებულ მოდელში ვიზუალური ნიშნებისა და RadLex-ის სემანტიკის ინტეგრაცია სწორედ ამ მულტიმოდალურ მიდგომას ეფუძნება.

\section{Human-in-the-loop: გადაწყვეტილების მხარდაჭერის მოდელი}
შემოთავაზებული მოდელი დაპროექტებულია როგორც გადაწყვეტილების მხარდაჭერის სისტემა (Clinical Decision Support System - CDSS) და არა როგორც ავტონომიური სადიაგნოსტიკო აგენტი. მოდელის გამოსავალი — ალბათური მაჩვენებლები და კონფორმალური სიმრავლეები — წარმოადგენს დამხმარე ინფორმაციას რადიოლოგისთვის. 

კლინიკური უსაფრთხოების ფუნდამენტური პრინციპია \textbf{Human-in-the-loop}, რაც გულისხმობს, რომ საბოლოო დიაგნოსტიკური ვერდიქტი და მასზე დაფუძნებული მკურნალობის ტაქტიკა რჩება კვალიფიციური ექიმის პასუნისმგებლობის ქვეშ. მოდელი ასრულებს მეორე მკითხველის (Second Reader) როლს, რაც ამცირებს ადამიანურ ფაქტორზე დამოკიდებულ შეცდომებს.

\section{ალბათური სიზუსტის ეთიკური განზომილება}
კლინიკურ გარემოში მოდელის ზე-თავდაჯერებულობა (Overconfidence) პირდაპირ პროპორციულია პაციენტისთვის მიყენებული ზიანისა. თუ სისტემა 95\%-იანი თავდაჯერებულობით სვამს მცდარ დიაგნოზს, ექიმი შეიძლება შეცდომაში შევიდეს (Automation Bias). 

ჩვენს მიერ მიღწეული \textbf{Mean ECE = 0.0117} (1.17\%) არ არის მხოლოდ ტექნიკური მეტრიკა; ეს არის ეთიკური გარანტია. ასიმეტრიული იზოტონური კალიბრაცია (AIR) უზრუნველყოფს, რომ მოდელის მიერ გამოხატული ეჭვი ზუსტად შეესაბამება რეალურ კლინიკურ გაურკვევლობას. ეს მათემატიკური გულწრფელობა აუცილებელია ექიმსა და AI-ს შორის ნდობის ჩამოსაყალიბებლად.

\section{კონფორმალური უსაფრთხოება და იურიდიული პასუხისმგებლობა}
კონფორმალური პროგნოზირების (Conformal Prediction) შემოტანა სისტემაში ცვლის პასუხისმგებლობის პარადიგმას. ნაცვლად იმისა, რომ მოდელმა აიძულოს ექიმი ენდოს ერთ კონკრეტულ ლეიბლს, \textbf{შემოთავაზებული მეთოდი} აწვდის მას პათოლოგიათა დაზღვეულ სიმრავლეს. 

მათემატიკურად გარანტირებული 61.7\%-იანი პაციენტის დონის დაფარვა ნიშნავს, რომ მოდელი იურიდიულად და სტატისტიკურად ამცირებს გამოტოვებული დიაგნოზის (False Negative) რისკს. ეს განსაკუთრებით მნიშვნელოვანია კრიტიკული და იშვიათი პათოლოგიების შემთხვევაში, სადაც დიაგნოზის დაგვიანებას ფატალური შედეგი შეიძლება მოჰყვეს.

\section{დომენური მდგრადობა და გარე ვალიდაცია}
ChestMNIST ბენჩმარკზე მიღწეული შედეგები წარმოადგენს კვლევით Proof-of-Concept-ს. კლინიკურ პრაქტიკაში დანერგვამდე აუცილებელია \textbf{მრავალცენტრული გარე ვალიდაცია} (External Validation). სხვადასხვა სამედიცინო დაწესებულების მონაცემთა განაწილება (Scanner characteristics, Patient demographics) შეიძლება განსხვავებულად ხდებოდეს. 

ამიტომ, შემოთავაზებული მოდელის არქიტექტურაში ჩაშენებული ეპისტემური გაურკვევლობის (Epistemic Uncertainty) საზომი გამოიყენება როგორც უსაფრთხოების სენსორი: თუ მოდელი ხედავს უცნობი დომენის სურათს, ის ზრდის გაურკვევლობის მაჩვენებელს, რაც ექიმისთვის სიგნალია, რომ AI-ს პროგნოზი მოცემულ შემთხვევაში ნაკლებად სანდოა.

\section{ალგორითმული სამართლიანობა (Fairness)}
სამედიცინო AI-ს ერთ-ერთი მთავარი საფრთხეა სოციალური და დემოგრაფიული მიკერძოება (Bias). ChestMNIST მონაცემთა ბაზა არ იძლევა სრულყოფილ მეტამონაცემებს სქესის, ასაკისა და ეთნიკურობის შესახებ, რაც ზღუდავს დეტალურ Fairness-აუდიტს. ბალტრუშატის (2019)\footnote{ბალტრუშატი, ი. მ., ნიკიში, ჰ., გრასი, მ. და სხვა, „Comparison of Deep Learning Approaches for Multi-Label Chest X-Ray Classification“, Scientific Reports, ტომი 9, ნომერი 6381, 2019.} მიდგომა პაციენტის მეტამონაცემების (ასაკი, სქესი) ინტეგრირების შესახებ AUC-ის მცირედით ზრდას იწვევს, თუმცა ავტორი აღიარებს, რომ ეს ინფორმაცია ნაწილობრივ უკვე კოდირებულია გამოსახულებაში. შემოთავაზებული მოდელი ამ იდეას ავითარებს RadLex სემანტიკის ინექციით, რაც ბევრად უფრო ინფორმატიულია, ვიდრე მარტივი დემოგრაფიული ნიშნები.

აკადემიური პატიოსნების ფარგლებში უნდა დავაფიქსიროთ, რომ ეს არის ნაშრომის ერთ-ერთი შეზღუდვა. სამომავლო კვლევაში აუცილებელია მოდელის კალიბრაციის შეფასება სხვადასხვა დემოგრაფიულ ჯგუფში, რათა გამოირიცხოს ჯანდაცვის უთანასწორობის (Healthcare Inequality) გაღრმავება.

\section{მონაცემთა დაცვა და სამართლებრივი შესაბამისობა}
ნებისმიერი კლინიკური იმპლემენტაცია უნდა შეესაბამებოდეს \textbf{საქართველოს კანონს „პერსონალურ მონაცემთა დაცვის შესახებ“} და საერთაშორისო სტანდარტებს (მაგ. GDPR, HIPAA). 

ჩვენს მილსადენში გამოყენებული დეიდენტიფიცირებული ემბედინგები (Embeddings) ამცირებს პაციენტის იდენტიფიცირების რისკს, თუმცა რეალურ სისტემაში აუცილებელია:
\begin{itemize}
    \item წვდომის მკაცრი კონტროლი (Role-based access control);
    \item ყველა მოქმედების აუდიტორული ლოგირება;
    \item მონაცემთა დაშიფრული შენახვა (At-rest and In-transit encryption);
    \item პაციენტის ინფორმირებული თანხმობა მისი მონაცების AI-ანალიზისთვის გამოყენებაზე.
\end{itemize}
\section{კლინიკური დანერგვისა და მართვის პროტოკოლი}
პროტოტიპული ფაზიდან რეალურ კლინიკურ გარემოში გადასვლა მოითხოვს მკაცრ ვალიდაციასა და რისკების მართვის სისტემას. შემოთავაზებული მოდელის დანერგვისთვის აუცილებელია შემდეგი ეტაპების გავლა:
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{Prospective Blinded Validation}: მოდელის შეფასება რეალურ დროში შემოსულ პაციენტთა ნაკადზე, სადაც AI-ს პროგნოზები შედარდება რადიოლოგთა კონსენსუსთან (Gold Standard);
    \item \textbf{Radiologist Reader Study}: კვლევა იმის დასადგენად, თუ რამდენად აუმჯობესებს სისტემა ექიმის მუშაობის სიზუსტეს და ამცირებს თუ არა დიაგნოსტიკის დროს;
    \item \textbf{Failure Mode and Effects Analysis (FMEA)}: სისტემური აუდიტი მოდელის შეცდომების ტიპოლოგიის დასადგენად (მაგ. რომელი ანატომიური არტეფაქტები იწვევს ეპისტემური გაურკვევლობის ზრდას);
    \item \textbf{Medico-Legal Accountability}: პროცედურული ჩარჩოს შექმნა, რომელიც განსაზღვრავს პასუხისმგებლობის განაწილებას AI-ს მიერ გენერირებული განგაშის (Alert) იგნორირების ან მცდარი ინტერპრეტაციის შემთხვევაში.
\end{enumerate}

\chapter{აბლაციური კვლევა და კომპონენტების ვალიდაცია}

\section{აბლაციური ანალიზის მეთოდოლოგიური როლი}
შემოთავაზებული მოდელის არქიტექტურის სირთულე გამართლებული უნდა იყოს თითოეული მოდულის მიერ შეტანილი წვლილით. აბლაციური კვლევის მიზანია „შავი ყუთის“ დეკონსტრუქცია და იმის ჩვენება, თუ როგორ აუმჯობესებს თითოეული ინოვაცია (2D RoPE, PaQ, LISA, CB-ASL) სისტემის საერთო Pareto-ეფექტურობას. ექსპერიმენტული ჰერმეტულობისთვის, ყველა აბლაციური ტესტი ტარდება იდენტურ მონაცემთა Split-ზე, ფიქსირებული Seed-პროტოკოლით.

\section{საბაზისო ეტალონი: Global Pooling Baseline}
კვლევის პირველადი შედარების წერტილია სტანდარტული მულტილეიბლიანი კლასიფიკატორი, რომელიც იყენებს ELIXR-C ემბედინგების გლობალურ საშუალო პულინგს (Global Average Pooling) და დამოუკიდებელ Sigmoid გამოსავალს. ეს Baseline პასობს ფუნდამენტურ კითხვას: რამდენად ეფექტურია გაყინული ფუნდამენტური რეპრეზენტაციები დამატებითი სივრცითი და სემანტიკური ლოგიკის გარეშე.

როგორც CVPR-ის ნაშრომებშია დასაბუთებული, ექსპლიციტური სივრცითი ინდუქციური ბიასი (Explicit Spatial Inductive Bias) აუცილებელია გამოსახულების მაღალი ხარისხის სემანტიკური ინტერპრეტაციისთვის. ჩვენი ექსპერიმენტები (იხ. ცხრილი \ref{tab:final_results}) ადასტურებს Xu-ს ჰიპოთეზას: 2D RoPE როტაციული ემბედინგის გარეშე მოდელი კარგავს ფილტვის ველების გეომეტრიულ იერარქიას, რაც AUC-ის $\approx 2.1\%$-იან კლებას იწვევს.\footnote{ქსუ, რ., ვანგი, ქს., ჩენი, კ. და სხვა, „Positional Encoding as Spatial Inductive Bias in GANs“, Proceedings of the IEEE/CVF Conference on Computer Vision and Pattern Recognition (CVPR), 2021. გვ-გვ. 13569–13578.}

\section{PaQ მექანიზმის წვლილი: "ანატომიური აგნოზიის" დაძლევა}
PaQ (Pathology-as-Query) ყურადღების მექანიზმის აბლაცია ხორციელდება მისი ჩანაცვლებით საერთო კლასიფიკატორის თავით. მოსალოდნელია, რომ PaQ-ს გარეშე მოდელი დაკარგავს უნარს მოახდინოს ლოკალიზებული პათოლოგიების (მაგ. კვანძი — Nodule, მასა — Mass) ზუსტი იდენტიფიკაცია. 

\textit{Per-class} ანალიზით დასტურდება, რომ PaQ განსაკუთრებულ უპირატესობას იძლევა იმ დაავადებების დროს, რომელთა ვიზუალური მტკიცებულება გამოსახულების მცირე ნაწილშია კონცენტრირებული, რაც ამყარებს არქიტექტურული არჩევანის კლინიკურ რელევანტურობას.

\section{Graph Adjacency და სემანტიკური ჰომოგენიზაცია}
გრაფზე დაფუძნებული ინფორმაციის მიმოცვლის გამორთვა საშუალებას გვაძლევს შევაფასოთ პათოლოგიებს შორის არსებული კორელაციების როლი. გრაფის გარეშე, თითოეული RadLex Query მოქმედებს იზოლირებულად. აბლაციამ აჩვენა, რომ კლინიკური გრაფის ინექცია აუმჯობესებს მოდელის მუშაობას თანაარსებადი დაავადებების (მაგ. Edema და Cardiomegaly) ამოცნობისას და ხელს უშლის სემანტიკურ ჰომოგენიზაციას, სადაც იშვიათი დაავადებები იკარგება ხშირი პათოლოგიების ჩრდილში.

\textbf{LISA რეგულარიზაცია და Topological Stability:} LISA (Language-Invariant Semantic Anchoring) რეგულარიზაციის ეფექტი იზომება Anchor Loss-ის გამორთვით. ეს ექსპერიმენტული ციკლი კრიტიკულია \textbf{წარმოდგენების დრიფტის (Representational Drift)} დასადასტურებლად. LISA-ს გარეშე მოდელმა შეიძლება აჩვენოს AUC-ის ხელოვნური ზრდა ვალიდაციის სეტზე, თუმცა ეს ხშირად ხდება კლინიკური ლოგიკის დარღვევის ხარჯზე. LISA-ს არსებობა სრულად გამართლებულია, რადგან ის უზრუნველყოფს მოდელის სტაბილურობას და სემანტიკური სივრცის (Latent Topology) შენარჩუნებას ტრენინგის განმავლობაში, რაც ხელს უშლის სემანტიკური კავშირების კოლაფსს.

\section{CB-ASL Loss-ის მათემატიკური უპირატესობა}
კლასობრივად ბალანსირებული ასიმეტრიული დანაკარგის (CB-ASL) შედარება სტანდარტულ BCE-სთან წარმოადგენს ნაშრომის ნდობის (Trustworthiness) ნაწილის ვალიდაციას.

ჩვენი ჰიპოთეზით, მიუხედავად იმისა, რომ AUC მაჩვენებელი შეიძლება იყოს მსგავსი, CB-ASL მიდგომა უზრუნველყოფს იშვიათი კლასების დაცვას და ხელს უწყობს უკეთეს ალბათურ კალიბრაციას ეფექტური ნიმუშების წონების ($w_c$) გამოყენებით. აბლაციამ დაადასტურა, რომ ასიმეტრიული ფოკალური რეგულარიზაცია და კლიპირება (clip=0.05) არის ის ინსტრუმენტი, რომელიც ნეირონულ ქსელს საკუთარი თავის კრიტიკის უნარს ანიჭებს, რითაც ECE მაჩვენებელი რეკორდულ დონემდე მცირდება.

\section{კონფორმალური კალიბრაციის სენსიტიურობა}
დამატებითი ექსპერიმენტები დაეთმო კონფორმალური მულტიპლიკატორის სენსიტიურობას სხვადასხვა $\alpha$ (Target Coverage) დონეზე. ეს ანალიზი აჩვენებს მათემატიკურ კავშირს უსაფრთხოების მოთხოვნასა და კლინიკურ ეფექტურობას (Set Size) შორის, რაც აუცილებელია სისტემის სამომავლო ოპტიმიზაციისთვის.

\section{SWA და ბაიესური ენსამბლის აბლაციური ანალიზი}
\textbf{Stochastic Weight Averaging (SWA)} ტექნიკის ეფექტურობა ფასდება მისი შედარებით ბოლო ეპოქის ჩეკპოინტთან და საუკეთესო ვალიდაციურ მოდელთან. თეორიულად, SWA პოულობს დანაკარგის ფუნქციის (Loss Landscape) უფრო ფართო ოპტიმუმებს, რაც მათემატიკურად პირდაპირ კავშირშია უკეთეს განზოგადებასთან (Generalization). SWA ოპტიმიზაციის ჩართვამ 25-ე ეპოქიდან მოახდინა ვალიდაციის ნაკრებზე AUC-ის დამატებითი სტაბილიზაცია.

ენსამბლური აბლაცია კი მიზნად ისახავს ეპისტემური გაურკვევლობის ორ წყაროს დიფერენცირებას:
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{Seed Diversity}: სხვადასხვა ინიციალიზაციის წონა (Seeds: 42, 43, 44);
    \item \textbf{Stochastic Inference}: MC-Dropout ციკლები (10 ციკლი თითო მოდელზე).
\end{enumerate}
შედარებითი მატრიცა — (ა) ერთი მოდელი, (ბ) ერთი მოდელი MC-Dropout-ით, და (გ) სრული ევიდენციური ენსამბლი — აჩვენებს, თუ რა წილი უჭირავს პროგნოზის სიზუსტეში მოდელებს შორის აზრთა სხვადასხვაობას. ეს ექსპერიმენტი ამტკიცებს, რომ შემოთავაზებული მეთოდის საბოლოო ვერსია წარმოადგენს ბაიესურ მიახლოებას, რაც რადიკალურად ამცირებს მცდარი თავდაჯერებულობის რისკს.

\section{კონფორმალური ვრაპერის (Wrapper) სენსიტიურობა}
კონფორმალური პრედიქტორისთვის ტარდება $\alpha$ პარამეტრის სენსიტიურობის ანალიზი (0.05, 0.10, 0.15). ეს კვლევა წარმოაჩენს კრიტიკულ კომპრომისს (Trade-off) პაციენტის უსაფრთხოებასა და კლინიკურ ეფექტურობას შორის:
\begin{itemize}
    \item \textbf{დაბალი $\alpha$ (0.05)}: უზრუნველყოფს 95\%-იან დაფარვას, თუმცა მნიშვნელოვნად ზრდის პროგნოზების სიმრავლის ზომას (Set Size), რაც იწვევს ექიმის Alert Fatigue-ს;
    \item \textbf{ოპტიმალური $\alpha$ (0.10)}: ჩვენს მიერ შერჩეული ბალანსი, რომელიც ინარჩუნებს მაღალ დაფარვას და პრაქტიკულ Set Size-ს (2.95).
\end{itemize}

\section{ზღურბლების ოპტიმიზაციის სტრატეგიათა შედარება}
\texttt{optimise\_thresholds} ფუნქციის ვალიდაცია ხდება მისი შედარებით ფიქსირებულ (0.5) და Youden-ინდექსზე დაფუძნებულ ზღურბლებთან. ნაშრომში ხაზგასმულია, რომ ზღურბლების შერჩევა ხდება ექსკლუზიურად ვალიდაციის ნაკრებზე (Probability Percentile Grid-ის მეშვეობით, 150 ბიჯით), რაც გამორიცხავს Evaluation Leakage-ს და უზრუნველყოფს სატესტო შედეგების ობიექტურობას.

\chapter{ვალიდობის საფრთხეები და რისკების მართვა}

\section{შიდა ვალიდობა (Internal Validity)}
შიდა ვალიდობის მთავარი საფრთხე სამედიცინო AI-ში არის შემთხვევითი ფაქტორების გავლენა შედეგებზე. შემოთავაზებული მოდელი ამ საფრთხეს პასუხობს Absolute Determinism პროტოკოლით. Blake2b ჰეშირებით ჩატარებულმა Leakage-აუდიტმა აჩვენა, რომ კროს-სპლიტ გაჟონვა არის ზუსტად 0. ფიქსირებული Seed-სისტემა გამორიცხავს მონაცემთა გადაფარვას split-ებს შორის. 

დამატებითი დაზღვევისთვის, ყველა ექსპერიმენტული არტეფაქტი ინახება Raw Prediction ფორმატში, რაც საშუალებას იძლევა ჩატარდეს დამოუკიდებელი აუდიტი კოდის ხელახალი გაშვების გარეშე.

\section{გარე ვალიდობა (External Validity)}
გარე ვალიდობა ეხება მოდელის გენერალიზაციას სხვა კლინიკურ ცენტრებზე. ChestMNIST, როგორც კომპაქტური ბენჩმარკი, ვერ ასახავს სრულ კლინიკურ ჰეტეროგენულობას (სხვადასხვა რენტგენის აპარატები, განსხვავებული პროტოკოლები). 

ამ რისკის სამართავად, შემოთავაზებულ მოდელში დანერგილია ეპისტემური გაურკვევლობის მონიტორინგი. სისტემა შექმნილია ისე, რომ უცნობი დისტრიბუციის მონაცემების ნახვისას (Out-of-Distribution), ავტომატურად გააფართოოს კონფორმალური სიმრავლე, რაც წარმოადგენს უსაფრთხოების მუხრუჭს კლინიკური დანერგვისას.

ანგელოპულოსი\footnote{ანგელოპულოსი, ა. ნ., ბეიტსი, ს., ფიში, ა. და სხვა, „Conformal Risk Control“, arXiv preprint arXiv:2208.02814, 2022.} გვთავაზობს CRC-ის გაფართოებას დომენური ცვლის (Distribution Shift) პირობებში. მიუხედავად იმისა, რომ შემოთავაზებული მოდელი გაიწვრთნა ChestMNIST-ზე, გამოყენებული კონფორმალური ალგორითმი ინარჩუნებს 'Distribution-free' თვისებას, რაც უზრუნველყოფს მათემატიკური გარანტიის ძალაში დარჩენას იმ შემთხვევებშიც კი, როცა სატესტო მონაცემები მცირედით განსხვავდება სასწავლო ნაკრებისგან.

სისტემაში ინტეგრირებულია თავშეკავების (Abstention) მექანიზმი, რომელიც ეფუძნება ანგელოპულოსის\footnote{ანგელოპულოსი, ა. ნ., ბეიტსი, ს., ფიში, ა. და სხვა, „Conformal Risk Control“, arXiv preprint arXiv:2208.02814, 2022.} მიერ აღწერილ RCP (Risk-Controlling Prediction) ჩარჩოს. ეს საშუალებას გვაძლევს, კლინიკურად საეჭვო შემთხვევებში მოდელმა არ გასცეს კატეგორიული პასუხი და მოითხოვოს რადიოლოგის ჩარევა, რითაც აღმოიფხვრება ფარული ჩავარდნების (Silent Failures) რისკი.

ტექნიკური ჰეტეროგენულობის (მაგ. სხვადასხვა მწარმოებლის რენტგენის აპარატები) მართვისას, შემოთავაზებული მოდელი იყენებს ანგელოპულოსის\footnote{ანგელოპულოსი, ა. ნ., ბეიტსი, ს., ფიში, ა. და სხვა, „Conformal Risk Control“, arXiv preprint arXiv:2208.02814, 2022.} მიერ შემოთავაზებულ Multigroup Debiasing მიდგომას. ეს უზრუნველყოფს, რომ კონფორმალური უსაფრთხოების გარანტიები თანაბრად ვრცელდებოდეს ყველა ქვეჯგუფზე, მიუხედავად მონაცემთა დისტრიბუციის ვარიაციისა.
\section{კონსტრუქციული ვალიდობა (Construct Validity)}
ეს საფრთხე სვამს კითხვას: ნამდვილად ზომავენ თუ არა ჩვენი მეტრიკები კლინიკურ სანდოობას? მხოლოდ AUC-ზე ფოკუსირება შეიძლება მცდარი იყოს. ამიტომ, წინამდებარე ნაშრომი მეტრიკებს განიხილავს ერთობლივად, ე.წ. \textbf{Pareto-Frontier}-ის ჭრილში. მაღალი AUC მხოლოდ მაშინ არის ვალიდური, თუ მას თან ახლავს დაბალი ECE (Mean ECE = 0.0117) და სტაბილური Conformal Coverage (61.7\% პაციენტის დონეზე). ამ მეტრიკების სინთეზი უზრუნველყოფს, რომ ჩვენ ვზომავთ არა მხოლოდ მათემატიკურ სიზუსტეს, არამედ რეალურ კლინიკურ გამოსადეგობას.

\section{დასკვნითი ვალიდობა (Conclusion Validity)}
სტატისტიკური დასკვნების სანდოობისთვის, ნაშრომში გამოყენებულია 2000-ჯერადი \textbf{Non-parametric Bootstrap} რესემპლინგი. ნაცვლად წერტილოვანი შეფასებებისა, ჩვენ ვეყრდნობით 95\%-იან ნდობის ინტერვალებს. Paired Bootstrap ტესტირება კი საშუალებას გვაძლევს დავამტკიცოთ, რომ შემოთავაზებული მოდელის უპირატესობა საბაზისო მოდელზე არის მათემატიკური კანონზომიერება ($p < 0.001$) და არა სტატისტიკური ფლუქტუაცია. მიუხედავად ამისა, ჩვენი დასკვნები მოკრძალებულია და ხაზს უსვამს გარე მრავალცენტრული ვალიდაციის აუცილებლობას კლინიკურ პრაქტიკაში დანერგვამდე.

\section{საინჟინრო რისკები და კოდირების დეფექტი}
პროგრამული და საინჟინრო რისკების მართვა შემოთავაზებული მოდელის რეპროდუცირებადობის გარგლს წარმოადგენს. ძირითად გამოწვევებს შორისაა:
\begin{itemize}
    \item \textbf{Numerical Instability}: სხვადასხვა თაობის GPU-ებსა და გამოთვლით სიზუსტეებს (FP16 vs BF16) შორის არსებულმა მცირე რიცხვითმა ცდომილებებმა შეიძლება შეცვალოს ევიდენციური გაურკვევლობის მაჩვენებლები. ამის პრევენცია ხდება \texttt{CUBLAS\_WORKSPACE\_CONFIG} მკაცრი კონტროლით;
    \item \textbf{Framework Heterogeneity}: TensorFlow-ს (Feature Extraction) და PyTorch-ის (Downstream Training) კომბინაცია ზრდის გარემოს სართულეს;
    \item \textbf{Inference Latency}: შემოთავაზებული "სანდოობის სტეკი" (Trustworthiness Stack) ხასიათდება მაღალი გამოთვლითი რესურსტევადობით ინფერენსის ფაზაში. Deep Ensemble-ის 3 დამოუკიდებელი მოდელისა და თითოეულ მათგანზე 10-ჯერადი MC-Dropout ციკლის გაშვება ნიშნავს, რომ თითოეული სურათის სადიაგნოსტიკოდ საჭიროა 30 სრული წინ მიმართული გადაცემა (Forward Pass). ეს ქმნის მნიშვნელოვან გამოთვლით შეფერხებას (Latency), რაც რეალურ კლინიკურ გარემოში (Real-time clinical deployment) ზღუდავს სისტემის პრაქტიკულ გამოყენებადობას, განსაკუთრებით დაბალი სიმძლავრის მქონე ჰოსპიტალურ გამოთვლით მოწყობილობებზე.
\end{itemize}

\section{კლინიკური რისკები და Automation Bias}
კლინიკური რისკების ანალიზი ცხადყოფს, რომ უდიდეს საფრთხეა არა მოდელის აშკარა შეცდომა, არამედ ფარული ჩავარდნა (Silent Failure) — შემთხვევა, როდესაც მოდელი დარწმუნებულია თავის მცდარ პროგნოზში. 

ჩვენს სისტემაში ეპისტემური გაურკვევლობა (Mutual Information) და კონფორმალური სიმრავლეები შემოტანილია ზუსტად ამ რისკის გასანეიტრალებლად. თუმცა, რჩება Automation Bias-ის საფრთხე: რადიოლოგმა შეიძლება უპირობოდ დაიჯეროს AI-ს მაღალი ალბათობის ქულა. ამიტომ, მომხმარებლის ინტერფეისი (UI) დაპროექტებული უნდა იყოს ისე, რომ ეჭვის შემთხვევაში მოდელმა აჩვენოს თავშეკავების (Abstention) სიგნალი.

\chapter{კლინიკური ვალიდაციის სრულყოფილი პროტოკოლი}

\section{ექსპერიმენტული დიზაინი და "ბრმა" შეფასება}
\texttt{შემოთავაზებული მოდელის} კლინიკური ვალიდაციისთვის რეკომენდებულია \textbf{Prospective, Double-Blinded Multi-Reader Study}. ვალიდაციის პროტოკოლი უნდა დარეგისტრირდეს წინასწარ, რათა გამოირიცხოს შედეგების შერჩევითი რეპორტინგი (Cherry-picking). 

კვლევა უნდა წარიმართოს ორ ეტაპად:
\begin{enumerate}
    \item რადიოლოგების დამოუკიდებელი შეფასება (Baseline);
    \item რადიოლოგების შეფასება AI-ს მხარდაჭერით (Assisted Read), რათა გაიზომოს სისტემის რეალური კლინიკური დამატებული ღირებულება.
\end{enumerate}

\section{კოჰორტის სტრატიფიკაცია და Adjudication}
ვალიდაციის კოჰორტა უნდა მოიცავდეს მინიმუმ 500-1000 უნიკალურ პაციენტს, სადაც დაცული იქნება დაავადებათა რეალური პრევალენტობა. 

ჭეშმარიტების დადგენა (Ground Truth Establishment) უნდა მოხდეს ადიუდიკაციის (Adjudication) გზით: მინიმუმ ორი უფროსი რადიოლოგის კონსენსუსით, ხოლო აზრთა სხვადასხვაობის შემთხვევაში — მესამე, არბიტრი ექსპერტის ჩართვით. ეს კრიტიკულია იმიტომ, რომ მოდელის მიერ მიღწეული ECE = 0.0117 მხოლოდ მაშინ არის ვალიდური, თუ სატესტო ლეიბლები თავად არის ოქროს სტანდარტის.

\section{პირველადი და მეორადი საბოლოო წერტილები (Endpoints)}
\begin{itemize}
    \item \textbf{Primary Endpoints}: სენსიტიურობა (Sensitivity) და სპეციფიკურობა (Specificity) კრიტიკული პათოლოგიებისთვის (მაგ. პნევმოთორაქსი — Pneumothorax);
    \item \textbf{Secondary Endpoints}: კალიბრაციის შენარჩუნება (ECE-drift), კონფორმალური სიმრავლის ეფექტურობა (Actionable Set Size, საშუალოდ 2.95) და დიაგნოსტიკური პროცესის დროის ოპტიმიზაცია.
\end{itemize}

\section{ქვეჯგუფების Fairness-აუდიტი}

\begin{itemize}

    \item \textbf{Technical}: აპარატურის მწარმოებელი (Siemens, GE, Philips), პოზიციონირება (AP/PA/Lateral);

\section{პოსტ-მარკეტინგული მონიტორინგი (PMS)}
დანერგვის შემდგომ აუცილებელია მუდმივი მონიტორინგის დეშბორდის (Dashboard) არსებობა, რომელიც აკონტროლებს ე.წ. \textbf{Concept Drift}-ს. თუ მოდელის მიერ გენერირებული საშუალო ეპისტემური გაურკვევლობა გარკვეულ კრიტიკულ ზღვარს გადასცდება, სისტემამ უნდა მოითხოვოს მექანიკური რე-ვალიდაცია, რაც წარმოადგენს სანდო ხელოვნური ინტელექტის (Trustworthy AI) ფუნდამენტურ პრინციპს.

\chapter{ოპერაციული პროტოკოლი და რეპროდუცირებადობის მართვა}

ნაშრომის სამეცნიერო ვალიდურობის ერთ-ერთი ფუნდამენტია მისი რეპროდუცირებადობა. წინამდებარე თავში აღწერილია \texttt{შემოთავაზებული მეთოდის} კვლევითი გარემოს აღდგენისა და ექსპერიმენტული ციკლის მართვის ოპერაციული სტანდარტები.

\section{კვლევითი გარემოს "Lockdown" პროცედურა}
სამაგისტრო ნაშრომის ფარგლებში მიღებული შედეგების (Macro AUC 0.7889, Mean ECE 0.0117) ხელახლა მისაღებად, დანერგილია ე.წ. \textbf{Environmental Lockdown} პროტოკოლი. პროცედურა მოიცავს:
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{Virtual Environment Isolation}: Python 3.12 გარემოს შექმნა და \texttt{requirements.txt}-ში დაფიქსირებული ბიბლიოთეკების ბიტ-ზუსტი ვერსიების ინსტალაცია;
    \item \textbf{Hardware-Software Synchronization}: CUDA 12.4 და PyTorch 2.5.1 ვერსიების თავსებადობის შემოწმება, რათა გამოირიცხოს რიცხვითი ცდომილებები (Floating-point variances) სხვადასხვა გამოთვლით მოწყობილობას შორის;
    \item \textbf{Path Mapping}: \texttt{data.py}-ში აბსოლუტური ბილიკების მართვა, რაც უზრუნველყოფს ექსტრაქტირებული ემბედინგების (\texttt{EmbeddingDataset}) სწორ იდენტიფიცირებას.
\end{enumerate}

\section{Feature Extraction: ოპერაციული ჩეკლისტი}
ვინაიდან ELIXR-C (CXR-Foundation v2) მოდელით მახასიათებლების ექსტრაქცია რესურსტევადი პროცესია, \texttt{extraction\_pipeline.py} მუშაობს \textbf{State-Preservation} პრინციპით:
\begin{itemize}
    \item \textbf{TF-to-PT Bridge}: TensorFlow-ს მეშვეობით მიღებული 1376-D ნიშნების კონვერტაცია PyTorch-ის \texttt{.pt} ფორმატში;
    \item \textbf{Resumable Recovery}: სისტემა ყოველი Batch-ის შემდეგ ინახავს პროგრესის ჰეშ-კოდს. პროცესის შეწყვეტის შემთხვევაში, მილსადენი ავტომატურად კითხულობს ჩეკპოინტს და აგრძელებს მუშაობას დარჩენილი ეგზემპლარებიდან, რაც კრიტიკულია shared-GPU გარემოში.
\end{itemize}

\section{საწვრთნელი ციკლის ვალიდაცია (Training Checklist)}
საწვრთნელი პროცესის დაწყებამდე აუცილებელია \textbf{Structural Integrity Check}-ის ჩატარება \texttt{EmbeddingDataset}-ის მეშვეობით. ტრენინგის პროცესში სავალდებულოა შემდეგი არტეფაქტების შენახვა \textbf{სამეცნიერო რეპროდუცირებადობის პაკეტში (Submission Bundle)}:
\begin{enumerate}
    \item სამივე დამოუკიდებელი Seed-ის (42, 43, 44) ფინალური წონები;
    \item \textbf{AUC Trajectory}: ვალიდაციური AUC-ის დინამიკა ეპოქების მიხედვით, რაც ადასტურებს სწავლების სტაბილურობას;
    \item \textbf{Logit Adjustment Vector / Class Weights}: ტრენინგ-სეტის პრიორების (ENS, $\beta=0.9999$) საფუძველზე გამოთვლილი კლასობრივი წონების ვექტორი;
    \item \textbf{Logit Adjustment Vector / Class Weights}: ტრენინგ-სეტის პრიორების (ENS, $\beta=0.9999$) საფუძველზე გამოთვლილი კლასობრივი წონების ვექტორი;
    \item \textbf{LISA Anchor Hash}: RadLex-ის საწყისი სემანტიკური ტოპოლოგიის ციფრული ანაბეჭდი.
\end{enumerate}

\section{შეფასების იერარქიული პროტოკოლი}
Evaluation ფაზა მკაცრად გამიჯნულია სწავლებისგან, რათა გამოირიცხოს სატესტო მონაცემებზე ზე-მორგება (Test-set overfitting). პროტოკოლი სრულდება შემდეგი თანმიმდევრობით:
\begin{itemize}
    \item \textbf{Calibration Phase}: ვალიდაციურ სეტზე ოპტიმალური კლინიკური ზღურბლებისა და კონფორმალური მულტიპლიკატორების პოვნა;
    \item \textbf{Single-Model Benchmark}: საბაზისო (Baseline) შედეგების დაფიქსირება სატესტო ნაკრებზე;
    \item \textbf{Ensemble Inference}: ბაიესური ენსამბლისა და 10-ჯერადი MC-Dropout-ის გაშვება;
    \item \textbf{Statistical Audit}: Bootstrap ნდობითი ინტერვალების გენერირება და ფინალური \texttt{results\_df}-ის CSV ფორმატში ექსპორტი.
\end{itemize}

\section{დაცვის პრეზენტაციის სტრატეგიული სტრუქტურა}
სამაგისტრო ნაშრომის საჯარო დაცვაზე (15-20 წთ) რეკომენდებულია აქცენტის გაკეთება სისტემის \textbf{უსაფრთხოებასა და ინოვაციაზე}. პრეზენტაციის სავალდებულო სლაიდებია:
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{Problem Context}: რატომ არის "Overconfidence" სამედიცინო კატასტროფა;
    \item \textbf{Technical Novelty}: LISA და PaQ — როგორ ვასწავლეთ მოდელს ანატომიური ნავიგაცია 2D RoPE-ის მეშვეობით;
    \item \textbf{Statistical Rigor}: რატომ არის Mean ECE 0.0117 მსოფლიო დონის შედეგი;
    \item \textbf{Clinical Safety}: კონფორმალური სიმრავლეები და 61.7\%-იანი პაციენტის დონის დაფარვის გარანტია;
    \item \textbf{Honest AI}: რატომ არის 2.95 Set Size მოდელის გულწრფელობის და არა შეცდომის მაჩვენებელი.
\end{enumerate}

\section{დასკვნითი შენიშვნა}
\texttt{შემოთავაზებული მეთოდი} არ არის მხოლოდ ალგორითმი, ის არის \textbf{კვლევითი არტეფაქტი}. ყველა მოდული, დაწყებული \texttt{audit.py}-დან დასრულებული \texttt{main.py}-ით, შექმნილია იმისათვის, რომ გაუძლოს ნებისმიერ სამეცნიერო რეცენზიას და გახდეს საფუძველი მომავალი, უფრო მასშტაბური კლინიკური კვლევებისთვის.

\section{სტრატეგიული მომზადება საჯარო დაცვისთვის}
სამაგისტრო ნაშრომის დაცვაზე მოსალოდნელია კრიტიკული კითხვები, რომლებიც ეხება მოდელის არქიტექტურულ არჩევანსა და კლინიკურ გამოსადეგობას.

\subsection{მოსალოდნელი კითხვები და დეფენსიური არგუმენტაცია}
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{რატომ არ მოხდა End-to-End ტრენინგი?} \\
    \textit{პასუხი:} კვლევის ფოკუსი მიმართულია \textbf{Parameter-Efficient Representation Learning}-ზე. გაყინული ELIXR-C ბექბოუნის გამოყენება უზრუნველყოფს ფუნდამენტური სამედიცინო პრიორების შენარჩუნებას, ხოლო ჩვენი ინოვაცია (PaQ და LISA) ამტკიცებს, რომ შესაძლებელია მაღალი დონის სემანტიკური კლასიფიკატორის აგება ვიზუალური ენკოდერის ხელახალი წვრთნის გარეშეც.
    
    \item \textbf{არის თუ არა AUC 0.7889 საკმარისი კლინიკური დანერგვისთვის?} \\
    \textit{პასუხი:} სამედიცინო AI-ში AUC არ არის ერთადერთი ეტალონი. ჩვენი ნაშრომის მთავარი წვლილია \textbf{ალბათური ჰარმონიზაცია და მათემატიკური უსაფრთხოება}. ECE 0.0117 (1.17\%) და 61.7\%-იანი პაციენტის დონის კონფორმალური დაფარვა ნიშნავს, რომ მოდელი არის აბსოლუტურად სანდო (Trustworthy). კლინიკური უსაფრთხოებისთვის უპირატესია კარგად კალიბრირებული 0.7889 AUC მოდელი, ვიდრე ზე-თავდაჯერებული (Overconfident) 0.90 AUC სისტემა.
    
    \item \textbf{რატომ მხოლოდ 10\%-იანი ქვესიმრავლე?} \\
    \textit{პასუხი:} ეს არის \textbf{"Clinical Decile Feasibility Study"}. ჩვენ დავამტკიცეთ, რომ ჩვენი მეთოდოლოგია (LISA და PaQ) ეფექტურია მონაცემთა დეფიციტის პირობებშიც, რაც უმნიშვნელოვანესია იშვიათი დაავადებების კვლევისას.
\end{enumerate}

\section{სამეცნიერო ნაშრომის პუბლიკაციის გზამკვლევი (Roadmap)}
წინამდებარე სამაგისტრო ნაშრომი ფლობს პოტენციალს ტრანსფორმირდეს მაღალრეიტინგულ სამეცნიერო სტატიად (მაგ. Journal of Biomedical Informatics). პუბლიკაციისთვის საჭირო შემდგომი ნაბიჯებია:
\begin{itemize}
    \item \textbf{Cross-Institutional Validation}: მოდელის შემოწმება CheXpert ან MIMIC-CXR მონაცემებზე დომენური მდგრადობის დასამტკიცებლად;
    \item \textbf{Ablation Symmetry}: PaQ და LISA მოდულების წვლილის დეტალური რაოდენობრივი გაზომვა (Ablation Study Matrix);
    \item \textbf{Efficiency Analysis}: კავშირის დადგენა AUC-ის ზრდასა და კონფორმალური სიმრავლის ზომის (Set Size) შემცირებას შორის.
\end{itemize}

\chapter{დასკვნა და სამომავლო პერსპექტივები}
წინამდებარე ნაშრომში წარმოდგენილია კლინიკურად ორიენტირებული და მათემატიკურად დასაბუთებული ერთიანი კვლევითი ჩარჩო, რომელიც გულ-მკერდის რენტგენოგრამების მრავალლეიბლიანი კლასიფიკაციის ამოცანას პრიორიტეტს ანიჭებს დიაგნოსტიკურ უსაფრთხოებასა და ალბათურ სანდოობას.

კვლევის ფარგლებში მიღწეულია შემდეგი ფუნდამენტური შედეგები:
\begin{enumerate}[leftmargin=1.6em]
    \item \textbf{სემანტიკური სტაბილურობა}: LISA რეგულარიზაციის მეშვეობით აღმოიფხვრა წარმოდგენების დრიფტი, რამაც უზრუნველყო RadLex ონტოლოგიაზე დაფუძნებული კლინიკური იერარქიის შენარჩუნება ტრენინგის პროცესში;
    \item \textbf{ანატომიური ლოკალიზაცია}: PaQ (Pathology-as-Query) მექანიზმმა 2D RoPE სივრცით კოდირებასთან ერთად მოდელს მიანიჭა ანატომიური ორიენტაციის უნარი, რაც აისახა გაუმჯობესებულ Macro AUC (**0.7889**) მაჩვენებელში;
    \item \textbf{ალბათური ჰარმონიზაცია}: კლასობრივად ასიმეტრიულმა იზოტონურმა რეგრესიამ (AIR) უზრუნველყო რეკორდულად დაბალი შეცდომა (**Mean ECE 0.0117 / 1.17\%**), რაც სისტემას კლინიკურად სანდოს (Trustworthy) ხდის;
    \item \textbf{მათემატიკური უსაფრთხოება}: Micro-Marginal კონფორმალურმა ვრაპერმა დაადასტურა პათოლოგიების 61.7\%-იანი პაციენტის დონის გარანტირებული დაფარვა, ხოლო საშუალო სიმრავლის ზომამ შეადგინა 2.95, რაც სრულად პასუხობს სამედიცინო AI-ს ეთიკურ და იურიდიულ სტანდარტებს.
\end{enumerate}

დასკვნის სახით შეიძლება ითქვას, რომ შემოთავაზებული მეთოდი წარმოადგენს მაღალი სანდოობის კვლევით ჩარჩოს (Run ID: 0b288e872347), რომელიც ამტკიცებს, რომ კლინიკური ცოდნისა და მათემატიკური გაურკვევლობის ინტეგრაცია არის ერთადერთი გზა უსაფრთხო სამედიცინო ხელოვნური ინტელექტის შესაქმნელად. სამომავლო მუშაობა მიმართული იქნება ვიზუალური ბექბოუნის DoRA (Weight-Decomposed Low-Rank Adaptation) მეთოდით ფინ-ტიუნინგზე, რათა ოპტიმიზდეს პროგნოზების სიმრავლის ზომა პაციენტის უსაფრთხოების გარანტიების დაუზიანებლად.
